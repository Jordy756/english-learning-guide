## Cortesía Básica

Las palabras mágicas que funcionan en cualquier contexto:

```javascript
const politeness = {
  request: "Please",           // Por favor
  gratitude: "Thank you",      // Gracias
  response: "You're welcome",  // De nada
  apology: "Sorry",            // Lo siento
  permission: "Excuse me"      // Disculpe / Perdón
};
```

**Variaciones en contexto:**

## Pedir Ayuda y Aclaraciones

Cuando no entiendes o necesitas ayuda:

<CardGrid>
  <Card title="Pedir repetición">
    - **Sorry?** (¿Perdón?)
    - **Pardon?** (¿Disculpe? - más formal)
    - **What was that?** (¿Qué fue eso?)
    - **Could you repeat that?** (¿Podrías repetir eso?)
    - **Say that again?** (¿Puedes decir eso otra vez?)
  </Card>
  
  <Card title="Pedir clarificación">
    - **I don't understand** (No entiendo)
    - **What do you mean?** (¿Qué quieres decir?)
    - **Can you explain that?** (¿Puedes explicar eso?)
    - **I'm not sure I follow** (No estoy seguro de seguirte)
    - **Could you clarify?** (¿Podrías aclarar?)
  </Card>
  
  <Card title="Pedir hablar más lento">
    - **Can you speak more slowly?** (¿Puedes hablar más despacio?)
    - **Could you slow down?** (¿Podrías ir más lento?)
    - **A bit slower, please** (Un poco más lento, por favor)
  </Card>
  
  <Card title="Pedir ayuda en inglés">
    - **Do you speak English?** (¿Hablas inglés?)
    - **Can you help me?** (¿Puedes ayudarme?)
    - **I need help** (Necesito ayuda)
    - **How do you say...?** (¿Cómo se dice...?)
  </Card>

**Ejemplos de diálogo:**

- Excuse me, can you help me? (Disculpe, ¿puede ayudarme?)
- Sorry, I don't understand. Can you repeat that? (Perdón, no entiendo. ¿Puedes repetir eso?)
- How do you say "biblioteca" in English? (¿Cómo se dice "biblioteca" en inglés?)

## Hacer Peticiones

```javascript
const makeRequest = (action, politeness = "standard") => {
  const formats = {
    direct: action,                    // Directo
    polite: `Can you ${action}?`,      // Cortés
    formal: `Could you ${action}?`,    // Formal
    veryFormal: `Would you ${action}?` // Muy formal
  };
  return formats[politeness];
};

// Ejemplos:
makeRequest("open the door", "polite");
// "Can you open the door?" (¿Puedes abrir la puerta?)

makeRequest("send me the file", "formal");
// "Could you send me the file?" (¿Podrías enviarme el archivo?)
```

**Patrones comunes:**

- Can you help me with this? (¿Puedes ayudarme con esto?)
- Could you pass me the salt? (¿Podrías pasarme la sal?)
- Would you mind closing the window? (¿Te importaría cerrar la ventana?)
- Please call me later (Por favor llámame más tarde)
- Let me know when you're ready (Avísame cuando estés listo)

<Aside type="tip" title="Regla para peticiones corteses">
**Más cortés:** "Could you..." y "Would you mind..." son más formales que "Can you...". Usa "please" para añadir cortesía a cualquier petición.
</Aside>

## Conversación Social

Frases para charlar y mantener la conversación:

```javascript
const smallTalk = {
  starters: [
    "How's your day going?",      // ¿Cómo va tu día?
    "What have you been up to?",  // ¿Qué has estado haciendo?
    "Any plans for the weekend?"  // ¿Planes para el fin de semana?
  ],
  
  reactions: [
    "That's interesting!",   // ¡Qué interesante!
    "Really?",               // ¿En serio?
    "No way!",               // ¡No puede ser!
    "That's awesome!",       // ¡Qué genial!
    "I see"                  // Ya veo
  ],
  
  fillers: [
    "You know...",     // Sabes...
    "Well...",         // Bueno...
    "Actually...",     // De hecho...
    "By the way...",   // Por cierto...
    "Anyway..."        // En fin...
  ]
};
```

**Ejemplos en conversación:**

- That's awesome! Tell me more (¡Qué genial! Cuéntame más)
- Really? I didn't know that (¿En serio? No sabía eso)
- By the way, have you seen John? (Por cierto, ¿has visto a John?)
- Anyway, I should get going (En fin, debería irme)
- That reminds me... (Eso me recuerda...)

## Errores Comunes

<Aside type="danger" title="Confundir Sorry con Excuse me">
**"Excuse me"** se usa para:
- Interrumpir educadamente: "Excuse me, can I ask a question?" ⛔ con Sorry
- Pedir paso: "Excuse me" (al pasar entre personas) ✅
- Llamar atención: "Excuse me, waiter!" ✅

**"Sorry"** se usa para:
- Disculparse: "Sorry I'm late" ✅
- Simpatía: "I'm sorry to hear that" ✅
- Pedir repetición (informal): "Sorry?" ✅
</Aside>

<Aside type="danger" title="Usar No problem en contextos formales">
- A: "Thank you!" / B: "No problem" (en contexto profesional) ⛔
- A: "Thank you!" / B: **"You're welcome"** (De nada - formal) ✅
- A: "Thank you!" / B: **"My pleasure"** (Es un placer) ✅
- A: "Thank you!" / B: **"Happy to help"** (Encantado de ayudar) ✅

"No problem" es muy casual para situaciones profesionales.
</Aside>

<Aside type="danger" title="Omitir please en peticiones directas">
- "Give me the pen" (suena como orden) ⛔
- **"Can you give me the pen, please?"** (¿Puedes darme el bolígrafo, por favor?) ✅
- **"Please give me the pen"** (Por favor dame el bolígrafo) ✅

Peticiones directas sin "please" pueden sonar bruscas. Siempre añade cortesía.
</Aside>

<Aside type="danger" title="Traducir literalmente de nada">
- "Of nothing" (traducción literal de "de nada") ⛔
- **"You're welcome"** (De nada - respuesta estándar) ✅
- **"No problem"** (Sin problema - casual) ✅
- **"Anytime"** (Cuando quieras - amigable) ✅

No traduzcas palabra por palabra, usa las expresiones naturales en inglés.
</Aside>
