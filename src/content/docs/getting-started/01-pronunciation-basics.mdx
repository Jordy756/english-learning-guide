La pronunciación en inglés es como configurar tu motor de texto a voz: necesitas mapear sonidos del español a nuevos sonidos del inglés. A diferencia del español donde cada letra tiene generalmente un sonido fijo, el inglés es más como JavaScript: la misma "variable" (letra) puede tener diferentes "valores" (sonidos) según el contexto.

Dominar la pronunciación básica te permitirá ser entendido y ganar confianza desde el primer día. No necesitas un acento perfecto, solo claridad en los sonidos clave.

## El Sistema de Sonidos

El inglés tiene aproximadamente 44 sonidos distintos (fonemas), mientras que el español tiene solo 24. Es como aprender nuevas funciones que no existen en tu lenguaje nativo:

```js
const spanishSounds = {
  vowels: 5,      // a, e, i, o, u (siempre igual)
  consonants: 19
};

const englishSounds = {
  vowels: 20,     // vocales + diptongos (varían según contexto)
  consonants: 24  // incluye sonidos nuevos como th, r, v
};
```

<Aside type="tip" title="Regla fundamental">
En inglés, la **ortografía ≠ pronunciación**. Debes aprender los sonidos independientemente de cómo se escriben las palabras.
</Aside>

## Sonidos que No Existen en Español

Estos son los "métodos nuevos" que debes agregar a tu repertorio:

<Tabs>
  <TabItem label="TH sonoro /ð/">

**Ejemplo:** this, that, mother  
**Cómo hacerlo:** Lengua entre los dientes, vibrar cuerdas vocales

```js
// Mapeo incorrecto (español)
const wrong = "dis" // ✗

// Mapeo correcto (inglés)
const correct = "ðɪs" // ✓ (th suena)
```

  </TabItem>

  <TabItem label="TH sordo /θ/">

**Ejemplo:** think, thank, nothing  
**Cómo hacerlo:** Lengua entre los dientes, sin vibrar cuerdas vocales

```js
// Mapeo incorrecto
const wrong = "sink" // ✗ (cambia significado)

// Mapeo correcto
const correct = "θɪŋk" // ✓ (th aspirado)
```

  </TabItem>

  <TabItem label="R americana /r/">

**Ejemplo:** red, car, word  
**Cómo hacerlo:** Lengua enrollada hacia atrás, sin tocar el paladar

```js
// Mapeo español (vibrante)
const wrong = "rojo" // ✗ (suena muy fuerte)

// Mapeo inglés (retroflexa)
const correct = "ɹɛd" // ✓ (suave, gutural)
```

  </TabItem>

  <TabItem label="V labial /v/">

**Ejemplo:** very, have, love  
**Cómo hacerlo:** Dientes superiores sobre labio inferior, vibrar

```js
// Error común: pronunciar como B
const wrong = "bery" // ✗

// Correcto: dientes sobre labio
const correct = "vɛri" // ✓
```

  </TabItem>

  <TabItem label="Schwa /ə/">

**Ejemplo:** about, banana, problem  
**Cómo hacerlo:** Vocal neutra, boca relajada (sonido más común en inglés)

```js
// Todas las vocales pueden reducirse a schwa:
const words = {
  about: "əˈbaʊt",    // primera 'a' = schwa
  banana: "bəˈnænə",  // primera y última 'a' = schwa
  problem: "ˈprɑbləm" // segunda sílaba = schwa
};
```

  </TabItem>
</Tabs>

## Acentuación: El Timing del Inglés

El inglés es un lenguaje de "stress-timing" (como un bucle que enfatiza elementos clave), mientras que el español es "syllable-timed" (iteración uniforme). Esto significa que en inglés, las sílabas acentuadas se alargan y las no acentuadas se reducen:

```js
// Español: todas las sílabas tienen igual duración
español.forEach(syllable => console.log(syllable)); // TA-TA-TA-TA

// Inglés: sílabas acentuadas son más largas
english.forEach((syllable, i) => {
  if (i === stressedIndex) {
    console.log(syllable.toUpperCase() + "---"); // TA---ta-ta-TA---
  } else {
    console.log(syllable.toLowerCase());
  }
});
```

**Patrón de énfasis:**
- **WA**ter (no "wa-TER")
- **COM**puter (no "com-pu-TER")
- im**POR**tant (no "IM-por-tant")

<Aside type="caution" title="Diccionarios y marcas de acento">
Los diccionarios marcan la sílaba acentuada con `ˈ` antes de ella:  
- `ˈwater` → "WA-ter"  
- `comˈputer` → "com-PU-ter"  
- `imˈportant` → "im-POR-tant"

Aprende a identificar este símbolo para pronunciar correctamente palabras nuevas.
</Aside>

## Vocales: Cortas vs Largas

A diferencia del español donde "a" siempre suena igual, en inglés la duración de la vocal cambia el significado:

<CardGrid>
  <Card title="Vocales Cortas">
**ship** /ɪ/ (barco)  
**cat** /æ/ (gato)  
**hot** /ɑ/ (caliente)  
**cut** /ʌ/ (cortar)
  </Card>

  <Card title="Vocales Largas">
**sheep** /i:/ (oveja)  
**cart** /ɑ:/ (carrito)  
**bought** /ɔ:/ (compró)  
**boot** /u:/ (bota)
  </Card>
</CardGrid>

```js
// La duración importa (como setTimeout vs setInterval)
const short = "ship"; // /ɪ/ - corto
const long = "sheep"; // /i:/ - largo (el : indica duración)

// Cambiar duración = cambiar significado
if (duration === "short") return "barco";
if (duration === "long") return "oveja";
```

## Consonantes Finales: No Omitir

En español, las consonantes finales a menudo se suavizan o desaparecen. En inglés, debes pronunciarlas claramente:

```js
// Español: tendencia a omitir final
const español = "amor" → "amo(r)" // r casi desaparece

// Inglés: la consonante final es crítica
const english = {
  "car": "kɑr",    // la R debe sonar
  "card": "kɑrd",  // la D debe sonar
  "cards": "kɑrdz" // la S/Z debe sonar
};
```

**Pares críticos:**
- cat /kæt/ vs. cab /kæb/ (gato vs. taxi)
- seat /sit/ vs. seed /sid/ (asiento vs. semilla)
- back /bæk/ vs. bag /bæg/ (espalda vs. bolsa)

## Errores Comunes

<Aside type="danger" title="Error #1: Pronunciar TH como T o D">
**Explicación:** El sonido "th" no existe en español y muchos lo reemplazan por "t" o "d".

Incorrecto: ~~"tink"~~ (think), ~~"dis"~~ (this)  
Correcto: **"think"** /θɪŋk/, **"this"** /ðɪs/ (lengua entre dientes)
</Aside>

<Aside type="danger" title="Error #2: V como B">
**Explicación:** En español, "v" y "b" suenan igual. En inglés, "v" requiere dientes sobre labio.

Incorrecto: ~~"bery good"~~ (very good), ~~"I hab"~~ (I have)  
Correcto: **"very good"** /ˈvɛri ɡʊd/, **"I have"** /aɪ hæv/ (dientes sobre labio inferior)
</Aside>

<Aside type="danger" title="Error #3: Pronunciar todas las letras como se escriben">
**Explicación:** El inglés tiene muchas letras mudas y combinaciones especiales.

Incorrecto: ~~"k-now"~~ (know), ~~"lis-ten"~~ (listen), ~~"cas-tle"~~ (castle)  
Correcto: **"no"** /noʊ/ (k muda), **"lissen"** /ˈlɪsən/ (t muda), **"cassel"** /ˈkæsəl/ (t muda)
</Aside>
