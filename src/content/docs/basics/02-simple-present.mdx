## üí° Pattern Matching (Reglas de Conjugaci√≥n)

```typescript
function conjugateToBe(pronoun: Pronoun, tense: Tense): string {
  if (tense === "present") {
    switch (pronoun) {
      case "I":
        return "am";
      case "you":
      case "we":
      case "they":
        return "are";
      case "he":
      case "she":
      case "it":
        return "is";
    }
  } else if (tense === "past") {
    switch (pronoun) {
      case "I":
      case "he":
      case "she":
      case "it":
        return "was";
      case "you":
      case "we":
      case "they":
        return "were";
    }
  }
}

// Uso
conjugateToBe("I", "present");      // "am"
conjugateToBe("she", "present");    // "is"
conjugateToBe("they", "past");      // "were"
```

:::tip[Regla de Oro]
**Presente:** `was` ‚Üí singular (`I`, `he`, `she`, `it`) | `were` ‚Üí plural + `you`

**Pasado:** `am` ‚Üí solo `I` | `is` ‚Üí singular | `are` ‚Üí plural + `you`
:::
