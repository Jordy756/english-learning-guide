## ‚ö†Ô∏è Exception Handling (Errores Comunes)

Al programar en ingl√©s, evita estos errores comunes que rompen la compilaci√≥n de la oraci√≥n.

### 1. Type Mismatch (Concordancia)

No puedes invocar un m√©todo que no pertenece al prototipo del sujeto.

```javascript

  console.log( I.is("happy") );

  // Error: undefined is not a function
  // Did you mean: He.is("my friend")?
```

### 2. Dependency Injection innecesaria (`Do/Does`)

El verbo *To Be* es independiente. No intentes inyectar dependencias auxiliares.

> **Regla de Oro:** Si el verbo principal es *To Be*, el operador `Do/Does` es **deprecated** (obsoleto/prohibido).
