# `toBe()` - Declaración de Estados

El verbo `to be` funciona como una **función de estado** que declara propiedades del sujeto.

## Sintaxis Básica

<Tabs>
  <TabItem label="Presente">
```javascript
const VerbToBe = {
  I: "am",
  you: "are",
  he: "is",
  she: "is",
  it: "is",
  we: "are",
  they: "are"
};

// Ejemplos de uso
console.log(`I ${VerbToBe.I} happy`);        // "I am happy"
console.log(`You ${VerbToBe.you} tired`);    // "You are tired"
console.log(`She ${VerbToBe.she} ready`);    // "She is ready"
console.log(`We ${VerbToBe.we} students`);   // "We are students"
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
const VerbToBe = {
  I: "was",
  you: "were",
  he: "was",
  she: "was",
  it: "was",
  we: "were",
  they: "were"
};

// Ejemplos de uso
console.log(`I ${VerbToBe.I} happy`);        // "I was happy"
console.log(`You ${VerbToBe.you} tired`);    // "You were tired"
console.log(`She ${VerbToBe.she} ready`);    // "She was ready"
console.log(`We ${VerbToBe.we} students`);   // "We were students"
```
  </TabItem>
</Tabs>

:::note[Regla importante]
Con `toBe()` **NO** se usan auxiliares `do/does/did` para preguntas y negaciones.
:::

## Operadores

### Negación (`not`)

<Tabs>
  <TabItem label="Presente">
```javascript
/**
 * Función Deny (Negar): Agrega el "not" después del verbo.
 * Reutiliza el objeto 'VerbToBe' definido anteriormente.
 */
const deny = (subject, state) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "am", "is", "are"
  console.log(`${subject} ${verb} not ${state}`); // 2. Imprimimos la negación
};

// Uso
deny("I", "sad");        // "I am not sad"
deny("she", "here");     // "She is not here"
deny("they", "ready");   // "They are not ready"
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
/**
 * Función Deny (Negar): Agrega el "not" después del verbo.
 * Reutiliza el objeto 'VerbToBe' definido anteriormente.

};

// Uso
deny("he", "happy");     // "He was not happy"
deny("we", "tired");     // "We were not tired"
```
  </TabItem>
</Tabs>

### Query (Preguntas)
<Tabs>
  <TabItem label="Presente">
```javascript
/**
 * Función Ask: Implementa la inversión de orden (Subject-Verb Swap)
 * Reutiliza el objeto 'VerbToBe' definido anteriormente.
 */ 
const ask = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "am", "is", "are"
  console.log(`${verb} ${subject} ${complement}?`); // 2. Imprimimos la pregunta
};

// Ejecutamos las consultas
ask("you", "sure about this"); // "Are you sure about this?"
ask("she", "coming to the party"); // "Is she coming to the party?"
ask("I", "late"); // "Am I late?"
```
  </TabItem>

ask("you", "sure about that"); // "Were you sure about that?"
ask("he", "at the meeting"); // "Was he at the meeting?"
ask("they", "on time"); // "Were they on time?"

## Debugging común

:::danger[Error común #1]
```typescript
// ❌ INCORRECTO: Usar do/does con toBe
do("you", "ready");

// ✅ CORRECTO: Usar la función de estado directa
are("you", "ready");
```
:::

:::danger[Error común #2]
```typescript
// ❌ INCORRECTO: El objeto "I" no tiene el método .is()
I.is("happy");     // TypeError: I.is is not a function

// ✅ CORRECTO: Usar el método correspondiente
I.am("happy");
```
:::
