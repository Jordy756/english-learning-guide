El verbo `to be` es el equivalente a definir quién eres ("I am a developer") o cómo estás ("I am tired"). Es la base de todo, como el `Hello World` del inglés.

## Estructura Básica (Afirmativo)

<Tabs>
  <TabItem label="Presente">
```javascript
const VerbToBe = {
  I: "am",
  you: "are",
  he: "is",
  she: "is",
  it: "is",
  we: "are",
  they: "are"
};

// Ejemplos de uso
console.log(`I ${VerbToBe.I} happy`);        // "I am happy"
console.log(`You ${VerbToBe.you} tired`);    // "You are tired"
console.log(`She ${VerbToBe.she} ready`);    // "She is ready"
console.log(`We ${VerbToBe.we} students`);   // "We are students"
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
const VerbToBe = {
  I: "was",
  you: "were",
  he: "was",
  she: "was",
  it: "was",
  we: "were",
  they: "were"
};

// Ejemplos de uso
console.log(`I ${VerbToBe.I} happy`);        // "I was happy"
console.log(`You ${VerbToBe.you} tired`);    // "You were tired"
console.log(`She ${VerbToBe.she} ready`);    // "She was ready"
console.log(`We ${VerbToBe.we} students`);   // "We were students"
```
  </TabItem>

:::note[Nota Importante]
El verbo **To Be** es especial: se manda solo. **NO** necesita ayuda de auxiliares (como `do`, `does`, o `did`) para hacer preguntas o negaciones.
:::

## Formas Negativa e Interrogativa

### Negación (Negative Form)

<Tabs>
  <TabItem label="Presente">
```javascript
/**
 * Función Deny (Negar): Agrega el "not" después del verbo.
 * Reutiliza el objeto 'VerbToBe' definido anteriormente.
 */
const deny = (subject, state) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "am", "is", "are"
  console.log(`${subject} ${verb} not ${state}`); // 2. Imprimimos la negación
};

// Uso
deny("I", "sad");        // "I am not sad"
deny("she", "here");     // "She is not here"
deny("they", "ready");   // "They are not ready"
```

### Preguntas (Question Form)

## Errores Comunes (Debugging)

:::danger[Bug #1: Usar auxiliares innecesarios]
Un error clásico es intentar usar `do` o `does` con el verbo To Be. Recuerda: `To Be` es independiente.

```javascript
// ❌ BUG: ¡Esto no compila en inglés!
do("you", "ready"); // Error: Auxiliar Innecesario

// ✅ FIX: Usa el verbo To Be directamente
are("you", "ready"); // Are you ready?
```
:::

:::danger[Bug #2: Conjugación incorrecta]
Cada sujeto tiene su propio verbo asignado. No puedes mezclarlos.

```javascript
// ❌ BUG: "I" no es compatible con "is"
I.is("happy");     // Error: Conjugación Incorrecta

// ✅ FIX: Usa la conjugación correcta
I.am("happy"); // I am happy
```
:::
