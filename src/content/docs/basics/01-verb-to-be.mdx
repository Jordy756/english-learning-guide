El verbo `to be` es la columna vertebral del inglés. Lo usamos para expresar identidad ("I am a developer"), estados ("I am tired"), ubicación ("She is here") y características ("They are tall"). Sin este verbo, no podrías presentarte ni describir el mundo que te rodea. Dominar `to be` es como aprender a caminar antes de correr: es tu primer paso hacia la fluidez en inglés.

## ¿Cómo se conjuga?

El verbo `to be` cambia su forma dependiendo de quién realiza la acción. A diferencia del español donde "estar" y "ser" son diferentes, en inglés ambos conceptos se expresan con `to be`. Aquí está la conjugación completa:

<Tabs>
  <TabItem label="Presente">
```javascript
const VerbToBe = {
  I: "am",
  you: "are",
  he: "is",
  she: "is",
  it: "is",
  we: "are",
  they: "are"
};
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
const VerbToBe = {
  I: "was",
  you: "were",
  he: "was",
  she: "was",
  it: "was",
  we: "were",
  they: "were"
};
```
  </TabItem>
</Tabs>

## Formas de uso

El verbo `to be` se comporta de manera diferente a otros verbos en inglés. Tiene sus propias reglas para formar oraciones afirmativas, negativas e interrogativas. A continuación, verás cómo construir cada tipo de oración correctamente.

### Afirmaciones

Para construir una oración afirmativa, simplemente coloca el verbo después del sujeto. La estructura es: **Sujeto + Verbo + Complemento**.

<Tabs>
  <TabItem label="Presente">
```javascript
const affirm = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "am", "is", "are"
  console.log(`${subject} ${verb} ${complement}`); // 2. Imprimimos la afirmación
};

// Ejemplos de uso
affirm("I", "happy");        // "I am happy"
affirm("you", "tired");      // "You are tired"
affirm("she", "ready");      // "She is ready"
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
const affirm = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "was", "were"
  console.log(`${subject} ${verb} ${complement}`); // 2. Imprimimos la afirmación
};

// Ejemplos de uso
affirm("I", "happy");        // "I was happy"
affirm("you", "tired");      // "You were tired"
affirm("she", "ready");      // "She was ready"
```
  </TabItem>
</Tabs>

### Negaciones

Para negar algo, añade la palabra `not` inmediatamente después del verbo. No necesitas palabras adicionales. La estructura es: **Sujeto + Verbo + not + Complemento**.

<Tabs>
  <TabItem label="Presente">
```javascript
const deny = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "am", "is", "are"
  console.log(`${subject} ${verb} not ${complement}`); // 2. Imprimimos la negación
};

// Ejemplos de uso
deny("I", "sad");        // "I am not sad"
deny("she", "here");     // "She is not here"
deny("they", "ready");   // "They are not ready"
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
const deny = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "was", "were"
  console.log(`${subject} ${verb} not ${complement}`); // 2. Imprimimos la negación
};

// Ejemplos de uso
deny("I", "sad");        // "I was not sad"
deny("he", "happy");     // "He was not happy"
deny("we", "tired");     // "We were not tired"
```
  </TabItem>
</Tabs>

### Preguntas

Para hacer una pregunta con `to be`, invierte el orden del sujeto y el verbo. El verbo va primero, luego el sujeto. La estructura es: **Verbo + Sujeto + Complemento + ?**

<Tabs>
  <TabItem label="Presente">
```javascript
const ask = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "am", "is", "are"
  console.log(`${verb} ${subject} ${complement}?`); // 2. Imprimimos la pregunta
};

// Ejemplos de uso
ask("you", "sure about this"); // "Are you sure about this?"
ask("she", "coming to the party"); // "Is she coming to the party?"
ask("I", "late"); // "Am I late?"
```
  </TabItem>

  <TabItem label="Pasado">
```javascript
const ask = (subject, complement) => {
  const verb = VerbToBe[subject]; // 1. Obtenemos "was", "were"
  console.log(`${verb} ${subject} ${complement}?`); // 2. Imprimimos la pregunta
};

// Ejemplos de uso
ask("you", "sure about that"); // "Were you sure about that?"
ask("he", "at the meeting"); // "Was he at the meeting?"
ask("they", "on time"); // "Were they on time?"
```
  </TabItem>
</Tabs>

<Aside title='Regla importante'>
El verbo **to be** no necesita verbos auxiliares como `do`, `does` o `did` para formar preguntas o negaciones. Simplemente inviertes el orden (para preguntas) o agregas `not` (para negaciones).

**Otros verbos sí necesitan auxiliares:**
- "Do you **like** pizza?" (necesita `do`)
- "Does she **play** tennis?" (necesita `does`)

**To be es independiente:**
- "**Are** you ready?" (sin auxiliar)
- "**Is** she here?" (sin auxiliar)
</Aside>

## Errores que debes evitar

<Aside type="danger" title="Bug #1: Conjugación incorrecta">
  Cada sujeto tiene su propia forma del verbo. No puedes intercambiarlas.
```javascript
  // ❌ BUG: "I" no es compatible con "is"
  console.log(`I ${VerbToBe.he} happy`);  // Resultado: "I is happy" (Incorrecto)

  // ✅ FIX: Usa la forma correcta del verbo
  console.log(`I ${VerbToBe.I} happy`);   // Resultado: "I am happy"
```
</Aside>

<Aside type="danger" title="Bug #2: Olvidar la inversión en preguntas">
  Recuerda que en las preguntas el verbo va primero, no el sujeto.
```javascript
  // ❌ BUG: El orden está invertido
  console.log(`You ${VerbToBe.you} ready?`); // "You are ready?" (Afirmación, no pregunta)

  // ✅ FIX: Invierte el orden correctamente
  console.log(`${VerbToBe.you} you ready?`); // "Are you ready?" (Pregunta correcta)
```
</Aside>
