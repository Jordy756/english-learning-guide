El **futuro con will** es como una función que se ejecuta en el momento: no sabías que ibas a hacerlo, pero de repente surge la situación y reaccionas. Lo usas para predicciones basadas en lo que piensas (no en evidencia), decisiones que tomas en el momento, promesas que haces y para ofrecer ayuda. Es directo, inmediato y universal para cualquier sujeto.

---

## Estructura del Futuro con Will

La estructura es idéntica para todos los sujetos: simplemente usas **will + verbo en forma base**, sin importar si el sujeto es **I**, **you**, **he**, **she**, **it**, **we** o **they**. No necesitas conjugar el verbo ni añadir ninguna terminación.

<Tabs>
  <TabItem label="Afirmativa">
    ```javascript
    const affirmative = {
      structure: "Sujeto + will + verbo en forma base",
      pattern: "igual para todos los sujetos",
      examples: [
        "I will help you",           // Te ayudaré
        "You will understand",       // Entenderás
        "He will arrive soon",       // Él llegará pronto
        "She will call later",       // Ella llamará después
        "It will rain tomorrow",     // Lloverá mañana
        "We will see",               // Veremos
        "They will come"             // Ellos vendrán
      ]
    };
    ```

    **Ejemplos:**
    - I **will study** tonight (Estudiaré esta noche)
    - The movie **will start** at 8 PM (La película comenzará a las 8 PM)
    - She **will be** a great doctor (Ella será una gran doctora)
    - We **will travel** to Europe next year (Viajaremos a Europa el próximo año)
    - They **will win** the game (Ellos ganarán el juego)
  </TabItem>
  
  <TabItem label="Negativa">
    ```javascript
    const negative = {
      structure: "Sujeto + will + not + verbo en forma base",
      contraction: "will not = won't",
      examples: [
        "I will not / won't forget",      // No olvidaré
        "You will not / won't fail",      // No fallarás
        "He will not / won't come",       // Él no vendrá
        "She will not / won't agree",     // Ella no aceptará
        "It will not / won't work",       // No funcionará
        "We will not / won't give up",    // No nos rendiremos
        "They will not / won't believe"   // No creerán
      ]
    };
    ```

    Para negar, añade **not** después de **will** o usa la contracción **won't**.

    **Ejemplos:**
    - I **won't tell** anyone (No le diré a nadie)
    - The train **won't arrive** on time (El tren no llegará a tiempo)
    - He **won't pass** the exam (Él no pasará el examen)
    - We **won't eat** meat (No comeremos carne)
    - They **won't accept** the offer (No aceptarán la oferta)
  </TabItem>
  
  <TabItem label="Pregunta">
    ```javascript
    const question = {
      structure: "Will + sujeto + verbo en forma base + ?",
      pattern: "invierte will y sujeto",
      examples: [
        "Will I need a ticket?",         // ¿Necesitaré un boleto?
        "Will you come?",                // ¿Vendrás?
        "Will he finish today?",         // ¿Terminará él hoy?
        "Will she like it?",             // ¿Le gustará a ella?
        "Will it rain?",                 // ¿Lloverá?
        "Will we have time?",            // ¿Tendremos tiempo?
        "Will they call us?"             // ¿Nos llamarán?
      ]
    };
    ```

    Para preguntar, pon **will antes del sujeto**.

    **Respuestas cortas:**
    - Will you help me? → Yes, I **will** / No, I **won't**
    - Will she come? → Yes, she **will** / No, she **won't**
    - Will they agree? → Yes, they **will** / No, they **won't**

    **Ejemplos:**
    - **Will** you **be** at home tonight? (¿Estarás en casa esta noche?)
    - **Will** it **snow** this winter? (¿Nevará este invierno?)
    - **Will** they **remember** us? (¿Se acordarán de nosotros?)
  </TabItem>
</Tabs>

<Aside type="tip" title="Contracciones en afirmativa">
En conversaciones informales, **will** se contrae con el sujeto: **I'll**, **you'll**, **he'll**, **she'll**, **it'll**, **we'll**, **they'll**. Estas contracciones son muy comunes en inglés hablado y escrito informal.
</Aside>

## Usos del Futuro con Will

<Tabs>
  <TabItem label="Predicciones">
    Usa **will** para hacer predicciones basadas en lo que **piensas u opinas**, sin evidencia visible.

    ```javascript
    const predictions = [
      "It will rain tomorrow",        // Lloverá mañana
      "You will love this movie",     // Te encantará esta película
      "He will be late",              // Él llegará tarde
      "The exam will be difficult",   // El examen será difícil
      "They will win the match"       // Ganarán el partido
    ];
    ```

    **Ejemplos:**
    - I think she **will succeed** (Creo que ella tendrá éxito)
    - The weather **will be** nice next week (El clima estará bonito la próxima semana)
    - This **will take** a long time (Esto tomará mucho tiempo)
    - Scientists believe robots **will replace** humans (Los científicos creen que los robots reemplazarán a los humanos)
  </TabItem>

  <TabItem label="Decisiones espontáneas">
    Usa **will** para decisiones que tomas **en el momento**, cuando algo surge y reaccionas inmediatamente.

    ```javascript
    const spontaneousDecisions = {
      situation: "The phone is ringing",
      reaction: "I'll answer it!",    // Lo contesto yo
      
      situation: "We need milk",
      reaction: "I'll buy some",      // Compraré un poco
      
      situation: "You look tired",
      reaction: "I'll help you"       // Te ayudaré
    };
    ```

    **Ejemplos:**
    - Someone at the door? I'**ll get** it (¿Alguien en la puerta? Yo abro)
    - I'**ll have** the pasta, please (Tomaré la pasta, por favor)
    - It's cold. I'**ll close** the window (Hace frío. Cerraré la ventana)
    - I'**ll call** you tomorrow (Te llamaré mañana)
  </TabItem>

  <TabItem label="Promesas y ofrecimientos">
    Usa **will** para hacer **promesas** (comprometerte a algo) o **ofrecer ayuda**.

    ```javascript
    const promisesAndOffers = {
      promises: [
        "I will be there",           // Estaré ahí
        "I won't tell anyone",       // No le diré a nadie
        "We will pay you back",      // Te pagaremos de vuelta
        "I will never forget this"   // Nunca olvidaré esto
      ],
      offers: [
        "I'll help you",             // Te ayudaré
        "I'll carry that",           // Yo cargo eso
        "I'll make dinner",          // Yo preparo la cena
        "I'll drive you home"        // Te llevaré a casa
      ]
    };
    ```

    **Ejemplos:**
    - I **will always love** you (Siempre te amaré)
    - I **won't be** late again, I promise (No llegaré tarde otra vez, lo prometo)
    - Don't worry, I'**ll fix** it (No te preocupes, lo arreglaré)
    - **Will** you hold this for me? (¿Me sostienes esto?)
  </TabItem>
</Tabs>

## Expresiones de Tiempo con Will

Estas expresiones te ayudan a ubicar la acción en el futuro:

```javascript
const timeExpressions = {
  specific: [
    "tomorrow",              // mañana
    "next week/month/year",  // próxima semana/mes/año
    "tonight",               // esta noche
    "soon",                  // pronto
    "later"                  // después/más tarde
  ],
  general: [
    "in the future",         // en el futuro
    "someday",               // algún día
    "one day"                // un día
  ],
  withTime: [
    "in 5 minutes",          // en 5 minutos
    "in 2 hours",            // en 2 horas
    "in a week"              // en una semana
  ]
};
```

**Ejemplos:**

- I will see you **tomorrow** (Te veré mañana)
- She will call **later** (Ella llamará después)
- They will arrive **in 10 minutes** (Llegarán en 10 minutos)
- We will visit you **next month** (Te visitaremos el próximo mes)
- Robots will fly **in the future** (Los robots volarán en el futuro)

## Errores Comunes

<Aside type="danger" title="Añadir -s o conjugar el verbo después de will">
Después de **will**, el verbo **siempre** va en forma base, sin importar el sujeto.

- He will **comes** tomorrow ⛔
- He will **come** tomorrow (Él vendrá mañana) ✅
- She will **goes** to the party ⛔
- She will **go** to the party (Ella irá a la fiesta) ✅
</Aside>

<Aside type="danger" title="Usar will para planes ya decididos">
Si ya planeaste algo antes de hablar, usa **going to**, no will.

- I **will go** to the dentist tomorrow (ya tenía cita) ⛔
- I**'m going to go** to the dentist tomorrow (Voy a ir al dentista mañana) ✅
- We **will travel** next month (ya compramos boletos) ⛔
- We**'re going to travel** next month (Vamos a viajar el próximo mes) ✅
</Aside>

<Aside type="danger" title="Olvidar won't para la negativa">
No uses **will no** o **not will**. La negación es **will not** o **won't**.

- I **will no go** ⛔
- I **not will go** ⛔
- I **will not go** / I **won't go** (No iré) ✅
</Aside>

<Aside type="danger" title="Confundir will con el verbo want">
**Will** se usa para hablar del futuro. **Want** significa querer y se usa en presente. No los mezcles.

- I **want** go tomorrow ⛔
- I **want to go** tomorrow (Quiero ir mañana) ✅
- I **will go** tomorrow (Iré mañana) ✅
</Aside>
