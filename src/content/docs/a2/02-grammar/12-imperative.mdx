El **imperativo** en inglés funciona como una función que se ejecuta sin recibir un sujeto explícito: el receptor siempre es "you", ya sea una persona o un grupo. Se usa para dar instrucciones, órdenes, consejos y advertencias. A diferencia del español, tiene una sola forma para singular y plural, y la estructura es siempre la misma.

---

## Estructura del Imperativo

<Tabs>
  <TabItem label="Afirmativo">
    El imperativo afirmativo usa el **verbo en forma base**, sin sujeto y sin auxiliares. Es directo y sin adornos.

    ```javascript
    const affirmative = {
      pattern: "Verbo base + complemento",
      examples: [
        "Open the door",    // Abre la puerta
        "Sit down",         // Siéntate
        "Save the file",    // Guarda el archivo
      ]
    };
    ```

    **Ejemplos:**

    - **Open** the window (Abre la ventana)
    - **Turn right** at the corner (Gira a la derecha en la esquina)
    - **Add** two cups of flour (Agrega dos tazas de harina)
    - **Restart** the application (Reinicia la aplicación)
    - **Read** the instructions carefully (Lee las instrucciones con cuidado)
  </TabItem>

  <TabItem label="Negativo">
    El imperativo negativo usa **Don't + verbo base**. La contracción "don't" (do not) es la forma estándar en todos los registros.

    ```javascript
    const negative = {
      pattern: "Don't + verbo base + complemento",
      examples: [
        "Don't run",       // No corras
        "Don't be late",   // No llegues tarde
        "Don't delete it", // No lo borres
      ]
    };
    ```

    **Ejemplos:**

    - **Don't run** in the hallway (No corras en el pasillo)
    - **Don't touch** the screen (No toques la pantalla)
    - **Don't be** nervous (No estés nervioso)
    - **Don't add** too much salt (No agregues demasiada sal)
    - **Don't close** the terminal yet (No cierres la terminal todavía)
  </TabItem>

  <TabItem label="Con 'please'">
    Agregar **please** al inicio o al final suaviza la orden y la convierte en una solicitud cortés. Ambas posiciones son correctas.

    ```javascript
    const polite = {
      atStart: "Please + verbo base",
      atEnd: "Verbo base + ..., please",
      negative: "Please don't + verbo base"
    };
    ```

    **Ejemplos:**

    - **Please open** the door (Por favor abre la puerta)
    - Sit down, **please** (Siéntate, por favor)
    - **Please read** the instructions (Por favor lee las instrucciones)
    - Wait a moment, **please** (Espera un momento, por favor)
    - **Please don't** interrupt (Por favor no interrumpas)
  </TabItem>
</Tabs>

## Usos del Imperativo

El imperativo aparece en situaciones muy concretas de la vida diaria y del trabajo técnico.

<CardGrid>
  <Card title="Instrucciones">
    Se usa para explicar pasos a seguir, como en manuales, tutoriales o documentación técnica. Cada paso es un imperativo.
  </Card>

  <Card title="Indicaciones">
    Esencial para orientar a alguien en el espacio: cómo llegar a un lugar, cómo navegar una interfaz.
  </Card>

  <Card title="Recetas">
    Cada paso de una receta es un imperativo afirmativo. El patrón es idéntico al de las instrucciones técnicas en un README.
  </Card>

  <Card title="Advertencias">
    Advierte sobre peligros o errores. Generalmente usa el imperativo negativo con "Don't" o el afirmativo con verbos como "Handle" o "Check".
  </Card>

  <Card title="Consejos">
    Ofrece recomendaciones directas sin rodeos. Más directo que "you should", pero igualmente válido en contextos informales.
  </Card>
</CardGrid>

<Aside type="note" title="Recetas e instrucciones técnicas comparten el mismo patrón">
Una receta en inglés y la documentación de una librería usan exactamente el mismo imperativo: "Preheat the oven" funciona igual que "Install the package". Si has leído algún README, ya conoces la lógica del imperativo en inglés.
</Aside>

**Ejemplos:**

- **Click** the button to continue (Haz clic en el botón para continuar)
- **Turn left** at the traffic light (Gira a la izquierda en el semáforo)
- **Preheat** the oven to 180 degrees (Precalienta el horno a 180 grados)
- **Don't expose** the battery to heat (No expongas la batería al calor)
- **Practice** a little every day (Practica un poco todos los días)

## Let's + verbo

**Let's** (contracción de "let us") se usa para hacer sugerencias que incluyen tanto al hablante como al oyente. No es una orden: es una invitación a hacer algo juntos.

```javascript
const suggestions = {
  affirmative: "Let's + verbo base",
  negative: "Let's not + verbo base",
  examples: {
    go: "Let's go",              // Vamos / Vámonos
    start: "Let's start over",   // Empecemos de nuevo
    skip: "Let's not skip it",   // No lo saltemos
  }
};
```

**Ejemplos:**

- **Let's go** to the meeting (Vamos a la reunión)
- **Let's start** from the beginning (Empecemos desde el principio)
- **Let's not** waste time (No perdamos el tiempo)
- **Let's try** a different approach (Intentemos un enfoque diferente)
- **Let's not** open that file yet (No abramos ese archivo todavía)

<Aside type="tip" title="Let's not vs. Don't — quién está incluido">
"Let's not" incluye al hablante en la sugerencia. "Don't" solo se dirige al oyente. "Let's not run" (no corramos, incluye a todos) es diferente de "Don't run" (no corras, solo tú).
</Aside>

## Errores comunes

<Aside type="danger" title="Agregar 'to' antes del verbo">
El imperativo usa el verbo en forma base directamente. Agregar "to" es un error que surge de confundirlo con el infinitivo.

- **To open** the door ⛔
- **Open** the door (Abre la puerta) ✅
</Aside>

<Aside type="danger" title="Incluir el sujeto 'you' explícito">
En inglés, el sujeto "you" del imperativo siempre está implícito. Escribirlo hace que la oración suene agresiva o enfática, no es la forma estándar.

- **You open** the door ⛔
- **Open** the door (Abre la puerta) ✅
</Aside>

<Aside type="danger" title="Usar 'No' en lugar de 'Don't'">
La negación del imperativo en inglés requiere "Don't". Usar "No" directamente es la traducción literal del español y no funciona en inglés.

- **No run** in the hallway ⛔
- **Don't run** in the hallway (No corras en el pasillo) ✅
</Aside>

<Aside type="danger" title="Escribir 'Do not be' como 'Don't to be'">
Con el verbo "be", el imperativo funciona igual que con cualquier otro verbo. No se agrega "to" en la forma negativa.

- **Don't to be** late ⛔
- **Don't be** late (No llegues tarde) ✅
</Aside>

## Práctica

<Aside type="tip" title="Ejercicio diario">
- Escribe una receta simple en inglés (tostadas, sándwich o pasta) usando al menos 6 imperativos afirmativos y 2 negativos como "Don't burn the toast"
- Describe en inglés cómo llegar desde tu casa a un lugar cercano usando solo imperativos: "Turn left", "Go straight two blocks", "Cross the street"
- Toma el README o la documentación de cualquier herramienta que uses y subraya todos los imperativos que encuentres
- Escribe 5 consejos de trabajo o estudio usando el imperativo afirmativo y 3 advertencias usando el negativo
- Escribe 4 sugerencias con "Let's" y 2 con "Let's not" sobre actividades que harías con un compañero de trabajo
</Aside>
