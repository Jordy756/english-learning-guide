Los **phrasal verbs** (verbos compuestos) son como métodos encadenados en programación: combinan un verbo base con una o más partículas (preposiciones o adverbios) para crear un significado completamente nuevo. Son extremadamente comunes en inglés hablado y escrito, especialmente en conversaciones cotidianas.

La clave para dominarlos es entender que el significado del phrasal verb rara vez es la suma literal de sus partes. Por ejemplo, "turn on" (encender) no significa literalmente "girar en" sino que tiene su propio significado.

---

## Estructura de los Phrasal Verbs

Un phrasal verb típicamente consta de dos o tres elementos. Piénsalo como métodos encadenados en programación, donde el verbo es una función que devuelve un objeto con métodos particle:

```javascript
// Métodos encadenados: verb().particle()
const turn = () => ({
  on: () => ({ 
    meaning: "encender", 
    example: "Turn on the TV",
  }),
  off: () => ({ 
    meaning: "apagar", 
    example: "Turn off the light",
  }),
  into: () => ({ 
    meaning: "convertirse en", 
    example: "The caterpillar turned into a butterfly"
  })
});

// Uso del método encadenado:
const phrasalVerb1 = turn().on();
// { meaning: "encender", example: "Turn on the TV" }

const phrasalVerb2 = turn().off();
// { meaning: "apagar", example: "Turn off the light" }

const phrasalVerb3 = turn().into();
// { meaning: "convertirse en", example: "The caterpillar..." }

// Esto representa exactamente cómo 
// funciona en inglés: verb + particle = phrasal verb
```

<Aside type="tip" title="Consejos para aprender phrasal verbs">
1. **Agrupa por verbo base**: Aprender todos los phrasal verbs con "turn" o "get" es más eficiente que aprenderlos al azar.

2. **Memoriza si es separable o no**: Esta característica determina cómo usarlos correctamente.

3. **Usa en contexto**: No memorices el significado aislado. Crea oraciones con situaciones reales.

4. **Lee y escucha**: Los phrasal verbs son muy comunes en películas, series y conversaciones. Exponerte a ellos es clave.

5. **Practica con pronombres**: Especialmente para los separables, practica con "it", "them", "me", etc.
</Aside>

## Phrasal Verbs Separables e Inseparables

La característica más importante es saber si puedes separar el verbo de la partícula:

<Tabs>
  <TabItem label="Separables">
    Los phrasal verbs separables permiten que el objeto se coloque entre el verbo y la partícula. De hecho, cuando el objeto es un pronombre, **DEBE** ir en el medio.

    ```javascript
    const separable = {
      structure: "verb + object + particle OR verb + particle + object",
      examples: [
        { phrase: "turn on the light", meaning: "enciende la luz" },
        { phrase: "turn the light on", meaning: "enciende la luz" },
        { phrase: "turn it on", meaning: "enciéndelo" }, // SIEMPRE separado
        { phrase: "put off the meeting", meaning: "aplaza la reunión" },
        { phrase: "put the meeting off", meaning: "aplaza la reunión" },
        { phrase: "put it off", meaning: "apázalo" }
      ]
    };
    ```

    **Phrasal verbs separables comunes:**
    - turn on/off
    - put on/off
    - take off
    - pick up
    - turn down
    - switch on/off
    - call off
    - give up
  </TabItem>

  <TabItem label="Inseparables">
    Los phrasal verbs inseparables mantienen el verbo y la partícula juntos siempre. El objeto nunca va en el medio.

    ```javascript
    const inseparable = {
      structure: "verb + particle + object",
      examples: [
        { phrase: "look for the keys", meaning: "busca las llaves" },
        { phrase: "look for them", meaning: "búscalas" },
        { phrase: "look the keys for", meaning: "❌ INCORRECTO" },
        { phrase: "get over the problem", meaning: "superé el problema" },
        { phrase: "run into an old friend", meaning: "me encontré con un viejo amigo" },
        { phrase: "go through the document", meaning: "revisa el documento" }
      ]
    };
    ```

    **Phrasal verbs inseparables comunes:**
    - look for
    - get over
    - run into
    - go through
    - run away from
    - look after
    - look at
  </TabItem>
</Tabs>

## Phrasal Verbs de 3 Partes

Algunos phrasal verbs están formados por **tres elementos**: verbo + dos partículas. Estos son particularmente interesantes porque combinan preposiciones o adverbios de formas únicas que crean significados completamente nuevos.

La estructura es: **verb + adverb/preposition + preposition**

```javascript
// Phrasal verbs de 3 partes (verb + 2 particles)
const threePartPhrasalVerbs = [
  { 
    phrasal: "put up with", 
    parts: "put + up + with",
    meaning: "tolerar, aguantar", 
    example: "How do you put up with that noise?",
    type: "Inseparable"
  },
  { 
    phrasal: "get along with", 
    parts: "get + along + with",
    meaning: "llevarse bien con", 
    example: "She gets along with everyone",
    type: "Inseparable"
  },
  { 
    phrasal: "run away with", 
    parts: "run + away + with",
    meaning: "escaparse con, dejarse llevar por", 
    example: "Don't let your imagination run away with you",
    type: "Inseparable"
  },
  { 
    phrasal: "keep up with", 
    parts: "keep + up + with",
    meaning: "mantenerse al día con", 
    example: "It's hard to keep up with technology",
    type: "Inseparable"
  }
];
```

<Aside type="caution" title="Los phrasal verbs de 3 partes nunca se separan">
Los phrasal verbs de 3 partes **SIEMPRE son inseparables**. El patrón es: verb + particle 1 + particle 2 + object. La estructura nunca se divide.

- Look forward your visit to ⛔
- Look forward to your visit ✅
</Aside>

## Errores Comunes

<Aside type="danger" title="Separar phrasal verbs inseparables">
Los inseparables NUNCA permiten que el objeto vaya en el medio:

~~Look for it the key~~ ⛔ 

**Look for the key** (Busca la llave) ✅

~~Run into him that problem~~ ⛔ 

**Run into him** (Me lo encontré) ✅ 
</Aside>

<Aside type="danger" title="No separar phrasal verbs separables con pronombres">
Cuando el objeto es un pronombre, el verbo DEBE separarse:

~~I turn on the light it~~ ⛔

**I turn it on** (Lo enciendo) ✅ 

~~Pick up them your room~~ ⛔ 

**Pick it up** (Recógelo) ✅ 
</Aside>

<Aside type="danger" title="Diferenciar phrasal verbs similares">
Algunos phrasal verbs similares pueden tener diferente uso:

- **take off** (separable): "Put on your shirt" → "Put it on"
- **get off** (inseparable): "Get off the bus" ("Get the bus off")
 
~~Take off your shoes off~~ ⛔

**Take off your shoes** ✅

~~Get off it the bus~~ ⛔

**Get off the bus** (siempre junto) ✅ 
</Aside>

<Aside type="danger" title="Uso incorrecto del pasado">
No todos los phrasal verbs cambian de forma en pasado de manera irregular:

~~I looked for yesterday~~ ⛔  

**I looked for it yesterday** ("look for" sigue siendo regular) ✅ 

~~She turned on already~~ ⛔ 

**She turned it on already** ("turn on" también es regular) ✅ 
</Aside>
