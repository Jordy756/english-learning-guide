El **presente simple** es el tiempo verbal más utilizado en inglés. Lo usas para hablar de rutinas y hábitos, hechos generales y verdades universales, gustos y preferencias, y horarios fijos. Es como una función que se ejecuta regularmente: describe lo que haces todos los días, lo que es verdad ahora, o lo que siempre funciona de cierta manera.

---

## Reglas para Tercera Persona

A diferencia del español, el presente simple en inglés tiene una particularidad importante: con los sujetos **he, she, it** (tercera persona singular), el verbo **cambia** añadiendo -s, -es, o -ies según su terminación.

<Tabs>
  <TabItem label="-s">
    La mayoría de verbos simplemente añaden **-s** al final.
    
    ```javascript
    const verbs = {
      work: "works",      // trabaja
      play: "plays",      // juega
      eat: "eats",        // come
      sleep: "sleeps",    // duerme
      run: "runs",        // corre
      like: "likes",      // gusta
      speak: "speaks"     // habla
    };
    ```
    
    También aplica cuando el verbo termina en **vocal + y**:
    
    ```javascript
    const verbs = {
      play: "plays",      // juega
      say: "says",        // dice
      buy: "buys",        // compra
      enjoy: "enjoys",    // disfruta
      pay: "pays"         // paga
    };
    ```
  </TabItem>
  
  <TabItem label="-es">
    Si el verbo termina en **-ch, -sh, -s, -x, -z, -o**, añade **-es**.
    
    ```javascript
    const verbs = {
      watch: "watches",   // mira
      wash: "washes",     // lava
      kiss: "kisses",     // besa
      fix: "fixes",       // arregla
      buzz: "buzzes",     // zumba
      go: "goes",         // va
      do: "does"          // hace
    };
    ```
  </TabItem>
  
  <TabItem label="-ies">
    Si el verbo termina en **consonante + y**, cambia la **y** por **i** y añade **-es**.
    
    ```javascript
    const verbs = {
      study: "studies",   // estudia
      try: "tries",       // intenta
      cry: "cries",       // llora
      fly: "flies",       // vuela
      carry: "carries",   // lleva
      worry: "worries"    // preocupa
    };
    ```
  </TabItem>
</Tabs>

<Aside type="tip" title="Truco de pronunciación">
Si puedes pronunciar el verbo + "s" fácilmente, usa **-s**. Si suena raro o difícil (como "watchs"), necesitas **-es**.
</Aside>

<Aside type="note" title="Verbos que nunca cambian">
Los **verbos modales** (can, could, would, should, may, might, must, will, shall) **nunca** añaden -s, -es o -ies, sin importar el sujeto:
- I **can** swim / She **can** swim ✅
- He **must** work / They **must** work ✅
- She **cans** swim / He **musts** work ⛔

Estos verbos tienen reglas especiales que aprenderás en [Verbos Modales](/a2/02-grammar/11-modal-verbs/).
</Aside>

## Estructura del Presente Simple

<Tabs>
  <TabItem label="Afirmativa">
    ```javascript
    const affirmative = {
      structure: "Sujeto + verbo (+ s/es)",
      examples: [
        "I work from home",        // Trabajo desde casa
        "You speak English",       // Hablas inglés
        "He plays guitar",         // Él toca guitarra
        "She studies math",        // Ella estudia matemáticas
        "It works perfectly",      // Funciona perfectamente
        "We live in Madrid",       // Vivimos en Madrid
        "They watch Netflix"       // Ellos ven Netflix
      ]
    };
    ```

    **Ejemplos:**
    - My brother **works** at a bank (Mi hermano trabaja en un banco)
    - The store **opens** at 9 AM (La tienda abre a las 9 AM)
    - Dogs **love** to play (A los perros les encanta jugar)
    - I **drink** coffee every morning (Tomo café todas las mañanas)
    - Sarah **teaches** English online (Sarah enseña inglés online)
  </TabItem>
  
  <TabItem label="Negativa">
    ```javascript
    const negative = {
      structure: "Sujeto + do not/does not + verbo en forma base",
      helpers: [
        "do not (don't)",     // para I, you, we, they
        "does not (doesn't)"  // para he, she, it
      ],
      examples: [
        "I don't work on Sundays",     // No trabajo los domingos
        "You don't like pizza",        // No te gusta la pizza
        "He doesn't play soccer",      // Él no juega fútbol
        "She doesn't study here",      // Ella no estudia aquí
        "It doesn't work",             // No funciona
        "We don't eat meat",           // No comemos carne
        "They don't watch TV"          // No ven televisión
      ]
    };
    ```

    **Ejemplos:**
    - My cat doesn't **like** water (A mi gato no le gusta el agua)
    - We don't **understand** the question (No entendemos la pregunta)
    - John doesn't **drink** alcohol (John no bebe alcohol)
    - I don't **speak** French (No hablo francés)
    - They don't **live** together (Ellos no viven juntos)

    <Aside type="note" title="Contracciones">
    En el habla cotidiana y escritura informal, es muy común usar las formas contraídas:
    - **do not** = don't
    - **does not** = doesn't
    </Aside>
  </TabItem>
  
  <TabItem label="Pregunta">
    ```javascript
    const question = {
      structure: "Do/Does + sujeto + verbo en forma base + ?",
      helpers: [
        "Do",   // para I, you, we, they
        "Does"  // para he, she, it
      ],
      examples: [
        "Do I know you?",              // ¿Te conozco?
        "Do you like music?",          // ¿Te gusta la música?
        "Does he work here?",          // ¿Trabaja él aquí?
        "Does she speak Spanish?",     // ¿Habla ella español?
        "Does it work?",               // ¿Funciona?
        "Do we need tickets?",         // ¿Necesitamos boletos?
        "Do they live nearby?"         // ¿Viven cerca?
      ]
    };
    ```

    **Ejemplos:**
    - Do you **play** any instrument? (¿Tocas algún instrumento?)
    - Does your sister **work** downtown? (¿Tu hermana trabaja en el centro?)
    - Do they **understand** English? (¿Ellos entienden inglés?)
    - Does the bus **stop** here? (¿El autobús para aquí?)
    - Do you **remember** me? (¿Te acuerdas de mí?)

    **Repuestas cortas:**
    - Yes, I **do** / No, I **don't**
    - Yes, she **does** / No, she **doesn't**
    - Yes, they **do** / No, they **don't**
  </TabItem>
</Tabs>

<Aside type="tip" title="Combinación con adverbios de frecuencia">
El presente simple se usa frecuentemente con [adverbios de frecuencia](/a2/01-basics/02-adverbs/#adverbios-de-frecuencia) como **always**, **usually**, **sometimes**, **never**, etc. para describir qué tan seguido haces algo.
</Aside>

## Errores Comunes

<Aside type="danger" title="Olvidar la -s en tercera persona">
En presente simple, los verbos con he, she o it siempre deben llevar una terminación especial (-s, -es o -ies).

- She **work** at home ⛔
- She **works** at home (Ella trabaja en casa) ✅
</Aside>

<Aside type="danger" title="Añadir -s cuando hay auxiliar">
Cuando usas los auxiliares do, does, don't o doesn't, el verbo principal siempre vuelve a su **forma base**, sin importar el sujeto.

- Does he **works** here? ⛔
- Does he **work** here? (¿Trabaja él aquí?) ✅
- She doesn't **likes** coffee ⛔
- She doesn't **like** coffee (A ella no le gusta el café) ✅
</Aside>

<Aside type="danger" title="Confundir presente simple con continuo">
El presente simple describe hábitos y rutinas establecidas, no acciones en progreso en este momento.

- I am **working** every day ⛔
- I **work** every day (Trabajo todos los días) ✅
</Aside>

<Aside type="danger" title="Aplicar incorrectamente las reglas de terminación">
Cuando un verbo termina en "y", debes revisar qué letra viene antes: si es consonante se cambia por -ies, si es vocal solo se añade -s.

- He **studys** English ⛔
- He **studies** English (Él estudia inglés) ✅
- She **plaies** piano ⛔
- She **plays** piano (Ella toca piano) ✅
</Aside>
