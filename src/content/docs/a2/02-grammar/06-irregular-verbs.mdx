Los **verbos irregulares** son aquellos que no siguen la regla -ed para formar el pasado simple. Son como excepciones en el código: no se comportan de manera predecible, pero son tan comunes que tienes que memorizarlos. Aunque representan solo un 10% de todos los verbos en inglés, incluyen los más usados en conversaciones diarias como **go → went**, **have → had** y **make → made**.

---

## ¿Por Qué Existen Verbos Irregulares?

Los verbos irregulares provienen del inglés antiguo y han conservado sus formas originales a lo largo de los siglos. Mientras que los verbos regulares adoptaron el patrón -ed más moderno, estos verbos mantuvieron sus transformaciones históricas.

```javascript
const verbEvolution = {
  modernPattern: {
    verb: "walk",
    past: "walked",    // regla moderna (-ed)
    type: "regular"
  },
  
  ancientPattern: {
    verb: "go",
    past: "went",      // forma antigua preservada
    type: "irregular"
  }
};
```

**Ejemplos:**

- I **go** to school every day → I **went** to school yesterday (Voy a la escuela cada día → Fui a la escuela ayer)
- She **has** a car now → She **had** a car before (Ella tiene un carro ahora → Ella tenía un carro antes)
- We **make** breakfast today → We **made** breakfast yesterday (Hacemos desayuno hoy → Hicimos desayuno ayer)
- They **see** the movie → They **saw** the movie last night (Ven la película → Vieron la película anoche)

<Aside type="note" title="Los verbos irregulares más comunes son los más usados">
Aunque parezca injusto, los verbos irregulares incluyen palabras fundamentales como **be** (was/were), **have** (had), **do** (did), **go** (went), **get** (got) y **make** (made). Esto significa que aprenderlos te ayudará a comunicarte en la mayoría de situaciones cotidianas.
</Aside>

## Los 40 Verbos Irregulares Esenciales

Estos verbos están organizados por patrones de transformación para facilitar su memorización:

<Tabs>
  <TabItem label="Sin cambio">
    Estos verbos **no cambian** en pasado simple. Son los más fáciles de recordar:
    
    ```javascript
    const noChange = {
      cut: "cut",      // cortar
      put: "put",      // poner
      let: "let",      // permitir/dejar
      set: "set",      // establecer
      cost: "cost",    // costar
      hit: "hit",      // golpear
      hurt: "hurt"     // herir/doler
    };
    ```

    **Ejemplos:**
    - I **cut** the paper yesterday (Corté el papel ayer)
    - She **put** the book on the table (Ella puso el libro en la mesa)
    - They **let** me go home early (Me dejaron ir a casa temprano)
    - The phone **cost** $500 (El teléfono costó $500)
    - He **hurt** his leg last week (Él se lastimó la pierna la semana pasada)
  </TabItem>
  
  <TabItem label="Cambio i → a">
    Estos verbos cambian la vocal **i** por **a** en el pasado:
    
    ```javascript
    const iToA = {
      begin: "began",     // comenzar/empezar
      drink: "drank",     // beber
      ring: "rang",       // sonar/timbrar
      sing: "sang",       // cantar
      swim: "swam",       // nadar
      sit: "sat"          // sentarse
    };
    ```

    **Ejemplos:**
    - The class **began** at 9 AM (La clase comenzó a las 9 AM)
    - I **drank** coffee this morning (Tomé café esta mañana)
    - The phone **rang** five minutes ago (El teléfono sonó hace cinco minutos)
    - We **sang** happy birthday (Cantamos feliz cumpleaños)
    - They **swam** in the ocean (Nadaron en el océano)
  </TabItem>
  
  <TabItem label="Patrón -ought/-aught">
    Estos verbos terminan en **-ought** o **-aught** en el pasado:
    
    ```javascript
    const oughtAught = {
      bring: "brought",   // traer
      buy: "bought",      // comprar
      catch: "caught",    // atrapar/agarrar
      fight: "fought",    // pelear
      teach: "taught",    // enseñar
      think: "thought"    // pensar
    };
    ```

    **Ejemplos:**
    - She **brought** her laptop (Ella trajo su laptop)
    - I **bought** a new phone (Compré un teléfono nuevo)
    - He **caught** the ball (Él atrapó la pelota)
    - We **thought** about the problem (Pensamos en el problema)
    - They **taught** us English (Nos enseñaron inglés)
  </TabItem>
  
  <TabItem label="Cambio vocal">
    Estos verbos cambian su vocal principal de maneras diversas:
    
    ```javascript
    const vowelChange = {
      come: "came",       // venir
      run: "ran",         // correr
      become: "became",   // convertirse
      eat: "ate",         // comer
      give: "gave",       // dar
      see: "saw",         // ver
      take: "took",       // tomar/llevar
      write: "wrote",     // escribir
      break: "broke",     // romper
      speak: "spoke",     // hablar
      drive: "drove",     // conducir/manejar
      get: "got",         // obtener/conseguir
      forget: "forgot"    // olvidar
    };
    ```

    **Ejemplos:**
    - They **came** to the party (Vinieron a la fiesta)
    - I **ran** 5 kilometers (Corrí 5 kilómetros)
    - She **ate** pizza for dinner (Ella comió pizza para la cena)
    - We **saw** a good movie (Vimos una buena película)
    - He **took** the bus (Él tomó el autobús)
  </TabItem>
  
  <TabItem label="Completamente diferentes">
    Estos verbos cambian totalmente y no siguen ningún patrón:
    
    ```javascript
    const totalChange = {
      be: "was/were",     // ser/estar
      do: "did",          // hacer
      go: "went",         // ir
      have: "had",        // tener
      make: "made",       // hacer
      say: "said",        // decir
      tell: "told",       // decir/contar
      find: "found",      // encontrar
      hear: "heard",      // oír/escuchar
      leave: "left",      // dejar/irse
      meet: "met",        // conocer/encontrarse
      pay: "paid",        // pagar
      send: "sent",       // enviar
      sleep: "slept",     // dormir
      stand: "stood",     // estar de pie
      understand: "understood", // entender
      win: "won"          // ganar
    };
    ```

    **Ejemplos:**
    - I **was** at home yesterday (Estuve en casa ayer)
    - She **did** her homework (Ella hizo su tarea)
    - We **went** to the beach (Fuimos a la playa)
    - They **had** a great time (Se la pasaron genial)
    - He **made** dinner for us (Él hizo la cena para nosotros)
  </TabItem>

## Errores Comunes

<Aside type="danger" title="Añadir -ed a verbos irregulares">
Los verbos irregulares no siguen la regla -ed. Debes memorizar sus formas específicas.

- I **goed** to the store ⛔
- I **went** to the store (Fui a la tienda) ✅
- She **maked** a cake ⛔
- She **made** a cake (Ella hizo un pastel) ✅
- They **buyed** tickets ⛔
- They **bought** tickets (Ellos compraron boletos) ✅
</Aside>

<Aside type="danger" title="Confundir verbos irregulares similares">
Algunos verbos irregulares se parecen pero tienen pasados diferentes. No los mezcles.

- I **bring** my book → I **bringed** it ⛔
- I **bring** my book → I **brought** it (Traje mi libro) ✅
- He **sing** well → He **sang** yesterday ✅
- He **sink** the boat → He **sank** the boat (no sang) ✅
</Aside>

<Aside type="danger" title="Usar el pasado en negaciones e interrogaciones">
Cuando usas **did** o **didn't**, el verbo vuelve a su forma base, incluso si es irregular.

- Did you **went** there? ⛔
- Did you **go** there? (¿Fuiste allá?) ✅
- She didn't **saw** me ⛔
- She didn't **see** me (Ella no me vio) ✅
- They didn't **had** time ⛔
- They didn't **have** time (No tenían tiempo) ✅
</Aside>

<Aside type="danger" title="Olvidar que be tiene dos formas en pasado">
El verbo **be** es especial: usa **was** para I/he/she/it y **were** para you/we/they.

- They **was** at home ⛔
- They **were** at home (Estaban en casa) ✅
- I **were** tired ⛔
- I **was** tired (Estaba cansado) ✅
</Aside>
