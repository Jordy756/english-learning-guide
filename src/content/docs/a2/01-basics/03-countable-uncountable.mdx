Los sustantivos en inglés se dividen en dos categorías fundamentales: **contables** e **incontables**. Es como clasificar datos en tipos discretos (puedes contar cada elemento) versus continuos (medibles pero no individuales). Esta distinción es crucial porque determina qué artículos y cuantificadores puedes usar, y muchos errores comunes de hispanohablantes vienen de no reconocer qué sustantivos son incontables en inglés.

---

## Tipos de Sustantivos

<Tabs>
  <TabItem label="Contables">

Los sustantivos **contables** son cosas que puedes contar individualmente. Tienen forma singular y plural.

```javascript
const countableNouns = {
  singular: "a book",      // un libro
  plural: "two books",     // dos libros
  canCount: true,          // puedes contar: 1, 2, 3...
  hasPlural: true          // tiene forma plural
};
```

**Características:**
- Puedes usar **a/an** con la forma singular
- Tienen forma **plural** (agrega -s, -es, o forma irregular)
- Puedes usar **números** (one, two, three...)
- Puedes usar **many** (muchos) y **few** (pocos)

**Categorías comunes:**

- **Personas**: person, child, friend, student, teacher, doctor
- **Animales**: dog, cat, bird, fish, horse, lion
- **Cosas**: book, car, phone, chair, table, pen
- **Lugares**: city, country, house, school, store, park
- **Comida**: apple, egg, cookie, sandwich, tomato, potato

**Ejemplos:**

- I have **three books** (Tengo tres libros)
- She bought **two apples** (Ella compró dos manzanas)
- There are **five people** waiting (Hay cinco personas esperando)
- We visited **many countries** (Visitamos muchos países)
- He owns **a car** and **a motorcycle** (Él tiene un carro y una motocicleta)

</TabItem>

  <TabItem label="Incontables">

Los sustantivos **incontables** son materiales, líquidos, o conceptos que **no puedes dividir en unidades individuales**.

```javascript
const uncountableNouns = {
  form: "water",           // agua
  canCount: false,         // no puedes decir "one water, two waters"
  hasPlural: false,        // no existe "waters" (en este contexto)
  needsContainer: true     // usa "a glass of water"
};
```

**Características:**
- **NO** puedes usar **a/an** directamente
- **NO** tienen forma plural
- **NO** puedes usar números directamente
- Usa **much** (mucho) y **little** (poco)
- Necesitas **contenedores o unidades** para contar

**Categorías comunes:**

- **Líquidos**: water, milk, juice, coffee, tea, oil, wine
- **Alimentos**: bread, rice, pasta, meat, cheese, butter, sugar
- **Materiales**: wood, paper, glass, metal, plastic, cotton
- **Abstractos**: information, advice, money, time, work, music
- **Colectivos**: furniture, luggage, equipment, traffic, weather

**Ejemplos:**

- I drink **water** every day (Tomo agua todos los días)
- She needs **information** (Ella necesita información)
- They bought **furniture** (Compraron muebles)
- We don't have **time** (No tenemos tiempo)
- He gave me good **advice** (Él me dio buen consejo)

<Aside type="caution" title="Incontables que confunden a hispanohablantes">
Estos son **incontables** en inglés pero contables en español: **information** (información), **advice** (consejo), **furniture** (muebles), **luggage** (equipaje), **homework** (tarea), **bread** (pan), **rice** (arroz).
</Aside>

</TabItem>
</Tabs>

## Uso de A/An

Los artículos **a/an** (un/una) solo se usan con sustantivos **contables singulares**. Nunca con incontables.

<Tabs>
  <TabItem label="Contables">

```javascript
const withArticles = {
  incorrect: [
    "a water",             // un agua (incorrecto)
    "an information",      // una información (incorrecto)
    "a money",             // un dinero (incorrecto)
    "an advice"            // un consejo (incorrecto)
  ],
  correct: [
    "a book",              // un libro
    "an apple",            // una manzana
    "a car",               // un carro
    "an idea"              // una idea
  ]
};
```

**Ejemplos:**

- I want **a sandwich** (Quiero un sándwich)
- She needs **an umbrella** (Ella necesita un paraguas)
- He's reading **a book** (Él está leyendo un libro)
- We saw **an elephant** (Vimos un elefante)

</TabItem>

  <TabItem label="Incontables ⛔">

```javascript
const withoutArticles = {
  incorrect: [
    "a water",             // un agua (incorrecto)
    "an information",      // una información (incorrecto)
    "a money",             // un dinero (incorrecto)
    "an advice"            // un consejo (incorrecto)
  ],
  correct: [
    "water",               // agua (sin artículo)
    "some water",          // algo de agua (con cuantificador)
    "a glass of water"     // un vaso de agua (con contenedor)
  ]
};
```

**Ejemplos:**

- I need **water** ⛔ (not "a water")
- I need **some water** ✅ (Necesito agua)
- She wants **information** ⛔ (not "an information")  
- She wants **some information** ✅ (Ella quiere información)

</TabItem>
</Tabs>

## Uso de Some y Any

**Some** (algo de/algunos) y **any** (algo de/ningún) se usan con sustantivos **contables en plural** e **incontables**.

<Tabs>
  <TabItem label="Some (Afirmativas)">

Usa **some** en oraciones **afirmativas** para indicar una cantidad indefinida.

```javascript
const someUsage = {
  countable: "I have some books",        // Tengo algunos libros
  uncountable: "I have some money"       // Tengo algo de dinero
};
```

**Ejemplos:**

- We need **some apples** (Necesitamos algunas manzanas) - contable
- They have **some water** (Tienen algo de agua) - incontable
- She bought **some chairs** (Ella compró algunas sillas) - contable
- I want **some information** (Quiero algo de información) - incontable

</TabItem>

  <TabItem label="Any (Negativas/Preguntas)">

Usa **any** en oraciones **negativas** y **preguntas**.

```javascript
const anyUsage = {
  question: "Do you have any books?",       // ¿Tienes algunos libros?
  negative: "I don't have any money"        // No tengo dinero
};
```

**Ejemplos:**

- Do you have **any eggs**? (¿Tienes huevos?) - pregunta, contable
- Is there **any milk**? (¿Hay leche?) - pregunta, incontable
- We don't have **any chairs** (No tenemos sillas) - negativa, contable
- I don't need **any help** (No necesito ayuda) - negativa, incontable

</TabItem>

  <TabItem label="Some en Preguntas (Ofrecimientos)">

Usa **some** en preguntas cuando **ofreces** algo o **pides** algo de manera cortés.

```javascript
const someInQuestions = {
  offering: "Would you like some coffee?",      // ¿Te gustaría café?
  requesting: "Can I have some water?"          // ¿Puedo tener agua?
};
```

**Ejemplos:**

- Would you like **some cookies**? (¿Te gustarían algunas galletas?)
- Can I have **some advice**? (¿Puedo recibir algún consejo?)
- Do you want **some help**? (¿Quieres ayuda?)

</TabItem>
</Tabs>

## Contenedores y Unidades

Para contar sustantivos incontables, usa **contenedores, porciones o unidades**. Es como envolver datos continuos en estructuras discretas.

```javascript
const containers = {
  pattern: "a/an + container + of + uncountable noun",
  example: "a glass of water"  // un vaso de agua
};
```

## Errores Comunes

<Aside type="danger" title="Usar a/an con sustantivos incontables">
- I need **a** water ⛔
- I need **some** water (Necesito agua) ✅
- She wants **an** information ⛔
- She wants **some** information (Ella quiere información) ✅

Los incontables NUNCA llevan **a/an**. Usa **some**, o un contenedor como "a glass of water".
</Aside>

<Aside type="danger" title="Pluralizar sustantivos incontables">
- I have two **breads** ⛔
- I have two **loaves of bread** (Tengo dos barras de pan) ✅
- We need three **furnitures** ⛔
- We need three **pieces of furniture** (Necesitamos tres muebles) ✅

Los incontables no tienen plural. Usa contenedores o unidades para contarlos.
</Aside>

<Aside type="danger" title="Tratar 'advice', 'information', 'furniture' como contables">
- Can you give me **an advice**? ⛔
- Can you give me **some advice**? (¿Puedes darme un consejo?) ✅
- I need **informations** ⛔
- I need **some information** (Necesito información) ✅

Estos sustantivos son incontables en inglés aunque sean contables en español.
</Aside>

<Aside type="danger" title="Usar 'any' en afirmativas">
- I have **any** money ⛔
- I have **some** money (Tengo dinero) ✅
- We bought **any** apples ⛔
- We bought **some** apples (Compramos manzanas) ✅

Usa **some** en afirmativas. **Any** es para negativas y preguntas.
</Aside>

<Aside type="danger" title="Confundir 'much' y 'many'">
- There are **much** people ⛔
- There are **many** people (Hay muchas personas) ✅
- I don't have **many** money ⛔
- I don't have **much** money (No tengo mucho dinero) ✅

**Many** con contables, **much** con incontables (esto se profundiza en el siguiente tema).
</Aside>

## Práctica

<Aside type="tip" title="Ejercicio diario">
- **Ejercicio de clasificación**: Clasifica 20 sustantivos como contables o incontables (libro, agua, consejo, silla, dinero, etc.)
- **Gap-fill**: Completa oraciones con a/an/some/any: "I need ___ water", "Do you have ___ books?", "She doesn't have ___ time"
- Transforma sustantivos incontables usando contenedores: "water" → "a glass of water", "bread" → "a loaf of bread"
- Lee un artículo en inglés y subraya todos los sustantivos incontables, verifica si se usan correctamente
- Practica con comida: haz una lista de compras usando some/any correctamente
- Memoriza los 10 incontables más problemáticos: water, money, information, advice, furniture, luggage, homework, bread, rice, hair
</Aside>
