Dominar días, meses y fechas es fundamental para la comunicación diaria en inglés: programar reuniones, hacer reservas, hablar sobre eventos o simplemente mencionar tu cumpleaños. A diferencia del español, los días y meses **siempre se escriben con mayúscula inicial**, y existen convenciones diferentes para expresar fechas según se trate de inglés británico o americano.

---

## Días de la semana

Los días de la semana en inglés son invariables (no cambian de forma) y **siempre se escriben con mayúscula**.

```javascript
const weekDays = [
  "Monday",    // Lunes
  "Tuesday",   // Martes
  "Wednesday", // Miércoles
  "Thursday",  // Jueves
  "Friday",    // Viernes
  "Saturday",  // Sábado
  "Sunday"     // Domingo
];
```

**Ejemplos:**

- I work from **Monday** to **Friday** (Trabajo de lunes a viernes)
- The meeting is on **Wednesday** (La reunión es el miércoles)
- We don't work on **Sundays** (No trabajamos los domingos)
- See you next **Tuesday**! (¡Nos vemos el próximo martes!)
- My birthday is on a **Saturday** this year (Mi cumpleaños cae en sábado este año)

<Aside type="tip" title="Preposición on para días específicos">
Usa **on** antes del día: "on Monday", "on Saturday". Para rangos usa **from... to**: "from Monday to Friday".
</Aside>

## Meses del año

Los meses también se escriben **siempre con mayúscula inicial** en inglés.

```javascript
const q1 = ["January",  "February", "March"];     // Enero, Febrero, Marzo
const q2 = ["April",    "May",      "June"];      // Abril, Mayo, Junio
const q3 = ["July",     "August",   "September"]; // Julio, Agosto, Septiembre
const q4 = ["October",  "November", "December"];  // Octubre, Noviembre, Diciembre
```

**Ejemplos:**

- My birthday is in **July** (Mi cumpleaños es en julio)
- We're traveling in **December** (Viajaremos en diciembre)
- Spring starts in **March** (La primavera empieza en marzo)
- I started working here in **January** (Empecé a trabajar aquí en enero)
- The conference is in **October** (La conferencia es en octubre)

<Aside type="tip" title="Preposición in para meses y años">
Usa **in** antes del mes: "in January", "in December". También se usa **in** para años: "in 2024".
</Aside>

## Expresar fechas

Las fechas en inglés tienen formatos diferentes según el contexto (británico vs. americano).

<Tabs>
  <TabItem label="Formato Británico">
    ```javascript
    const number = "15/03/2024";
    const formal = `${day}th ${month} ${year}`; // 15th March 2024 (15 de marzo de 2024)
    const spoken = `the ${day}th of ${month}`;  // the fifteenth of March (el 15 de marzo)
    ```
    
    El formato británico sigue el orden: **día → mes → año**
  </TabItem>
  
  <TabItem label="Formato Americano">
    ```javascript
    const number = "03/15/2024";
    const formal = `${month} ${day}th, ${year}`; // March 15th, 2024 (15 de marzo de 2024)
    const spoken = `${month} ${day}th`;          // March fifteenth (15 de marzo)
    ```
    
    El formato americano sigue el orden: **mes → día → año**
  </TabItem>
  
  <TabItem label="Números Ordinales">
    ```javascript
    const ordinals = {
      irregular: ["1st", "2nd", "3rd"], // primero, segundo, tercero
      regular:   ["4th", "5th", "6th"], // cuarto, quinto, sexto
      compound:  ["21st", "32nd"]       // vigésimo primero, trigésimo segundo
    };
    ```
    
    Los ordinales se usan para los días del mes
  </TabItem>
</Tabs>

**Ejemplos:**

- Today is Monday, **January 15th** (Hoy es lunes 15 de enero)
- The deadline is **March 31st** (La fecha límite es el 31 de marzo)
- I was born on **June 5th**, 1990 (Nací el 5 de junio de 1990)
- The event is on Friday, **December 20th** (El evento es el viernes 20 de diciembre)
- We're meeting on the **10th of May** (Nos reunimos el 10 de mayo)
- It's the **first** of April (Es el primero de abril)

## Expresiones útiles con tiempo

<Tabs>
  <TabItem label="Relativas">
    Expresiones de tiempo relativas al día de hoy.

    ```javascript
    const relativeTime = [
      "yesterday", "today", "tomorrow",     // ayer, hoy, mañana
      "last week", "this week", "next week", // la semana pasada, esta, la próxima
      "last year", "next month"             // el año pasado, el próximo mes
    ];
    ```

    **Ejemplos:**

    - She was here **yesterday** (Ella estuvo aquí ayer)
    - I have a meeting **tomorrow** (Tengo una reunión mañana)
    - We will start the project **next week** (Empezaremos el proyecto la próxima semana)
  </TabItem>

  <TabItem label="Frecuencia">
    Para hablar de qué tan seguido ocurre una acción.

    ```javascript
    const frequencyTime = [
      "every day", "every Monday",  // todos los días, todos los lunes
      "on weekends",                // los fines de semana
      "once a month",               // una vez al mes
      "twice a year"                // dos veces al año
    ];
    ```

    **Ejemplos:**

    - I drink coffee **every day** (Bebo café todos los días)
    - We play tennis **on weekends** (Jugamos al tenis los fines de semana)
    - She travels to Europe **twice a year** (Ella viaja a Europa dos veces al año)
  </TabItem>

  <TabItem label="Partes del día">
    Para especificar el momento del día. Nota que "night" usa "at" y no "in the".

    ```javascript
    const partsOfDay = [
      "in the morning",   // por la mañana
      "in the afternoon", // por la tarde / de tarde
      "in the evening",   // por la tarde / al anochecer
      "at night"          // por la noche
    ];
    ```

    **Ejemplos:**

    - He exercises **in the morning** (Él hace ejercicio por la mañana)
    - They arrived **in the afternoon** (Ellos llegaron por la tarde)
    - I read a book **at night** (Leo un libro por la noche)
  </TabItem>
</Tabs>

## Errores comunes

<Aside type="danger" title="No usar mayúsculas en días y meses">
Los días de la semana y meses siempre se escriben con mayúscula inicial en inglés, a diferencia del español.

- I'll see you on **monday** in **july** ⛔
- I'll see you on **Monday** in **July** (Te veré el lunes en julio) ✅
</Aside>

<Aside type="danger" title="Confundir preposiciones de tiempo">
Usa `in` para meses y años enteros, pero usa `on` para días específicos (fechas exactas o días de la semana).

- My birthday is **at** May ⛔
- The meeting is **in** Monday ⛔
- My birthday is **in** May (Mi cumpleaños es en mayo) ✅
- The meeting is **on** Monday (La reunión es el lunes) ✅
</Aside>

<Aside type="danger" title="Confusión con formato numérico de fechas">
El formato numérico puede causar confusión entre contextos británico (día/mes) y americano (mes/día).

- The event is on **03/05/2024** (ambiguo: ¿3 de mayo o 5 de marzo?) ⛔
- The event is on **March 5, 2024** (El evento es el 5 de marzo de 2024 - claro) ✅
</Aside>

<Aside type="danger" title="Olvidar números ordinales al hablar de fechas">
Los días del mes siempre se expresan con números ordinales al hablar (first, second, third...), no con cardinales (one, two, three...).

- Today is October **twenty-five** ⛔
- Today is October **twenty-fifth** (Hoy es veinticinco de octubre) ✅
</Aside>

## Práctica

<Aside type="tip" title="Ejercicio diario">
- Practica diciendo la fecha completa cada mañana: "Today is [day], [month] [ordinal], [year]"
- Deletrea en voz alta los días de la semana y meses del año escribiéndolos con mayúscula
- Escribe 5 fechas importantes para ti (cumpleaños, aniversarios) en formato británico y americano
- Pregunta y responde en voz alta: "When is your birthday?", "What's the date today?"
- Practica expresiones de tiempo: "I have a meeting next Monday", "I traveled last December"
</Aside>
