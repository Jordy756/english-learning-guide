Los saludos son como las funciones de inicialización en programación: establecen el tono de la conversación y demuestran respeto por el contexto social. En inglés, el nivel de formalidad varía significativamente según la situación, desde un "Hey" casual hasta un "Good morning" profesional, por lo que dominar estas variaciones te permite adaptarte con confianza a cualquier entorno, ya sea informal o corporativo.

---

## Saludos por Nivel de Formalidad

<Tabs>
  <TabItem label="Informal">

```js
const informalGreetings = {
  // Saludos
  basic: [
    "Hi", // Hola
    "Hey", // Hola (más casual)
    "Hello" // Hola (puede ser informal o neutral según el tono)
  ],
  casual: [
    "What's up?", // ¿Qué hay?
    "How's it going?" // ¿Cómo va?
  ],
  slang: [
    "Sup?", // ¿Qué hay?
    "Yo!" // ¡Ey!
  ],
  
  // Respuestas típicas
  responses: [
    "Hey!", // ¡Hola!
    "Not much", // Nada
    "Good, you?" // Bien, ¿y tú?
  ]
};
```

**Diálogos típicos:**
  - A: "Hey! How's it going?" (¡Hola! ¿Cómo va?)
  - B: "Good! How about you?" (¡Bien! ¿Y tú?)

  - A: "What's up?" (¿Qué hay?)
  - B: "Not much, just chilling" (Nada, aquí relajado)

  </TabItem>

  <TabItem label="Neutral">

```js
const neutralGreetings = {
  // Saludos estándar
  general: [
    "Hello", // Hola (puede ser neutral o formal según el tono)
    "Hi there", // Hola (un poco más formal que "Hi")
    "Good morning" // Buenos días (formal, pero común en el trabajo)
  ],
  questions: [
    "How are you?", // ¿Cómo estás?
    "How are you doing?" // ¿Cómo te va?
  ],
  
  // Respuestas educadas
  responses: [
    "I'm fine, thanks. And you?", // Estoy bien, gracias. ¿Y tú?
    "Good, thanks. How are you?" // Bien, gracias. ¿Cómo estás?
  ]
};
```

**Diálogos típicos:**
  - A: "Good morning! How are you?" (¡Buenos días! ¿Cómo estás?)
  - B: "I'm fine, thank you. How are you?" (Estoy bien, gracias. ¿Y tú?)

  </TabItem>

  <TabItem label="Formal">

```js
const formalGreetings = {
  // Saludos profesionales
  standard: [
    "Good morning", // Buenos días
    "Good afternoon" // Buenas tardes
  ],
  veryFormal: [
    "How do you do?", // ¿Cómo está usted? (respuesta igual a la pregunta)
    "It's a pleasure" // Es un placer (usado en presentaciones formales)
  ],
  
  // Respuestas formales
  responses: [
    "How do you do?", // Respuesta igual a la pregunta
    "The pleasure is mine", // El placer es mío
    "I'm very well, thank you" // Estoy muy bien, gracias
  ]
};
```

<Aside type="note" title="How do you do es una fórmula de cortesía">
"How do you do?" es MUY formal y se responde con "How do you do?" (no es realmente una pregunta sobre tu estado).
</Aside>

  </TabItem>
</Tabs>

<Aside type="tip" title="Regla de contexto">
Usa el nivel de formalidad según:
- **Informal:** Amigos, familia, personas de tu edad
- **Neutral:** Situaciones cotidianas, tiendas, conocidos
- **Formal:** Trabajo, entrevistas, personas mayores, primera vez
</Aside>

## Saludos por Momento del Día

El saludo cambia según la hora, pero estos son específicos y más formales que "Hi":

```js
const timeBasedGreetings = {
  morning: "Good morning",     // Buenos días (5:00 AM - 12:00 PM)
  afternoon: "Good afternoon", // Buenas tardes (12:00 PM - 6:00 PM)
  evening: "Good evening",     // Buenas noches (6:00 PM - 10:00 PM)
  night: null // NO existe "Good night" como saludo
};

const greet = (hour) => {
  if (hour >= 5 && hour < 12) return "Good morning"; // Buenos días
  if (hour >= 12 && hour < 18) return "Good afternoon"; // Buenas tardes
  if (hour >= 18 && hour < 22) return "Good evening"; // Buenas noches
  return "Hello"; // Hola (si es muy tarde, usa saludo neutral)
};
```

<Aside type="caution" title='"Good night" NO es un saludo'>
**"Good night"** es una DESPEDIDA, no un saludo. Se usa al irte a dormir o al final del día.

- "Good night" (al llegar a las 8 PM) ⛔
- **"Good evening"** (al llegar), **"Good night"** (al irte) ✅
</Aside>

## Respuestas a "How are you?"

Esta pregunta es cortesía social, no siempre espera una respuesta detallada:

<CardGrid>
  <Card title="Respuestas Positivas">
    - **"I'm fine, thank you"** (Estoy bien, gracias)
    - **"Good, thanks"** (Bien, gracias)
    - **"Great! And you?"** (¡Genial! ¿Y tú?)
    - **"Pretty good"** (Bastante bien)
    - **"Couldn't be better"** (No podría estar mejor)
  </Card>
  <Card title="Respuestas Neutrales">
    - **"Not bad"** (Nada mal)
    - **"Can't complain"** (No me quejo)
    - **"Same as always"** (Como siempre)
    - **"Alright"** (Bien)
    - **"Hanging in there"** (Ahí voy / Sobreviviendo)
  </Card>
  <Card title="Respuestas Honestas">
    - **"I'm okay"** (Estoy bien)
    - **"Could be better"** (Podría estar mejor)
    - **"A bit tired"** (Un poco cansado)
    - **"Not great, honestly"** (No muy bien, honestamente)
  </Card>
  <Card title="Devolver la pregunta">
    Siempre puedes devolver la pregunta:

    - **"And you?"** (¿Y tú?)
    - **"How about you?"** (¿Qué tal tú?)
    - **"What about you?"** (¿Y tú?)
  </Card>
</CardGrid>

## Presentaciones

Cuando conoces a alguien por primera vez:

<Tabs>
  <TabItem label="Informal">

```js
const informalIntro = {
  self: "I'm John / My name is John", // Soy John / Mi nombre es John
  nice: "Nice to meet you", // Encantado de conocerte
  response: "Nice to meet you too" // Encantado de conocerte también
};
```

**Diálogo:**
  - A: "Hi! I'm Sarah" (¡Hola! Soy Sarah)
  - B: "Hey! I'm Mike. Nice to meet you" (¡Hola! Soy Mike. Encantado de conocerte)
  - A: "Nice to meet you too!" (¡Encantada de conocerte también!)

  </TabItem>

  <TabItem label="Formal">

```js
const formalIntro = {
  self: "My name is John Smith", // Mi nombre es John Smith
  title: "I'm Dr./Mr./Ms. Johnson", // Soy Dr./Sr./Sra. Johnson
  pleasure: "It's a pleasure to meet you", // Es un placer conocerle
  response: "The pleasure is mine" // El placer es mío
};
```

**Diálogo:**
  - A: "Good morning. My name is Robert Chen" (Buenos días. Mi nombre es Robert Chen)
  - B: "Good morning, Mr. Chen. I'm Dr. Williams. It's a pleasure to meet you" (Buenos días, Sr. Chen. Soy el Dr. Williams. Es un placer conocerle)
  - A: "The pleasure is mine" (El placer es mío)

  </TabItem>

  <TabItem label="Con Contexto">

Añade información sobre ti:

```js
const introWithContext = (name, role, place) => {
  return `Hi, I'm ${name}. I'm a ${role} at ${place}`;
};

console.log(introWithContext("Ana", "developer", "Google"));
// "Hi, I'm Ana. I'm a developer at Google" - Hola, soy Ana. Soy desarrolladora en Google
```

**Ejemplos:**
  - "I'm Carlos. I'm a student at MIT" (Soy Carlos. Soy estudiante en MIT)
  - "My name is Lisa. I work in marketing" (Mi nombre es Lisa. Trabajo en marketing)
  - "I'm Tom from the IT department" (Soy Tom del departamento de IT)

  </TabItem>
</Tabs>

## Despedidas

Las despedidas también varían según el nivel de formalidad:

<Tabs>
  <TabItem label="Informales">

```js
const informalGoodbyes = [
  "Bye",              // Adiós
  "See you",          // Nos vemos
  "See you later",    // Nos vemos luego
  "Take care",        // Cuídate
  "Catch you later",  // Te veo luego
  "Later",            // Luego
  "Peace"             // Paz (muy informal)
];
```

  </TabItem>

  <TabItem label="Neutrales">

```js
const neutralGoodbyes = [
  "Goodbye",              // Adiós
  "Have a good day",      // Que tengas un buen día
  "Have a nice weekend",  // Que tengas un buen fin de semana
  "See you soon",         // Nos vemos pronto
  "Talk to you later",    // Hablamos luego
  "Take it easy"          // Tómatelo con calma
];
```

  </TabItem>

  <TabItem label="Formales">

```js
const formalGoodbyes = [
  "Goodbye",                  // Adiós
  "Have a pleasant day",      // Que tenga un día agradable
  "It was nice meeting you",  // Fue un placer conocerle
  "Until next time",          // Hasta la próxima
  "Best regards",             // Saludos cordiales
  "Good night"                // Buenas noches (al final del día/noche)
];
```

  </TabItem>

## Errores Comunes

<Aside type="danger" title="Usar Good night como saludo">
- "Good night! How are you?" (al llegar a las 9 PM) ⛔
- **"Good evening! How are you?"** (al llegar) ✅
- **"Good night!"** (al irte o ir a dormir) ✅

"Good night" es exclusivamente una despedida para el final del día.
</Aside>

<Aside type="danger" title="Responder literalmente a How do you do">
- "I'm doing fine, thanks" ⛔
- **"How do you do?"** (misma respuesta) ✅

"How do you do?" es una fórmula muy formal que funciona como saludo recíproco, no como pregunta real. Esta frase es poco común en conversación moderna.
</Aside>

<Aside type="danger" title="No devolver la pregunta How are you">
- A: "How are you?" / B: "Fine" (y silencio) ⛔
- A: "How are you?" / B: **"Good, thanks. And you?"** ✅
- A: "How are you?" / B: **"Fine. How about you?"** ✅

En inglés es descortés no preguntar por la otra persona después de responder.
</Aside>

## Práctica

<Aside type="tip" title="Ejercicio diario">
- Saluda a 3 personas diferentes en el día usando distintos niveles de formalidad (informal/neutral/formal)
- Practica frente al espejo cambiando el saludo según la hora: "Good morning" (mañana), "Good afternoon" (tarde), "Good evening" (noche)
- Escribe dos diálogos breves: uno para una situación formal (entrevista) y otro informal (amigos)
- Grábate pronunciando despedidas y asegúrate de no confundir "Good evening" con "Good night"
- Responde a "How are you?" de 3 formas diferentes y siempre devuelve la pregunta
</Aside>
