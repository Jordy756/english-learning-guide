El inglés usa las mismas 26 letras del español, pero cada una tiene un nombre diferente en inglés. Conocer su pronunciación correcta es esencial para deletrear tu nombre, email o dirección en cualquier situación.

---

## Las 26 Letras

El alfabeto inglés tiene la misma estructura que el español, pero cada letra se pronuncia de forma única:

```javascript
const alphabet = {
  total: 26,
  vowels: ["A", "E", "I", "O", "U"],  // vocales
  consonants: 21,                     // consonantes
  origin: "Latin"                     // origen latino
};
```

<Aside type="tip" title="Regla clave">
**No traduzcas** el nombre de las letras del español. Por ejemplo, "B" no se dice "be" como en español, sino "bi" /biː/ en inglés.
</Aside>

## Alfabeto

Pronunciación estándar de cada letra del alfabeto:

```javascript
const englishAlphabet = {
  A: "ei",        B: "bi",        C: "si",
  D: "di",        E: "i",         F: "ef",
  G: "yi",        H: "eich",      I: "ai",
  J: "yei",       K: "kei",       L: "el",
  M: "em",        N: "en",        O: "ou",
  P: "pi",        Q: "kiu",       R: "ar",
  S: "es",        T: "ti",        U: "iu",
  V: "vi",        W: "dabel iu",  X: "eks",
  Y: "uai",       Z: "zi" // "zed" en UK
};
```

**Ejemplos:**

- **BOOK** se deletrea "bi - ou - ou - kei" (libro)
- **CODE** se deletrea "si - ou - di - i" (código)
- **EMAIL** se deletrea "i - em - ei - ai - el" (correo)
- **HOME** se deletrea "eich - ou - em - i" (hogar)

<Aside type="note" title="Vocales importantes">
Las vocales tienen nombres que coinciden con su "sonido largo":
- **A** = "ei" /eɪ/ (no "a")
- **E** = "i" /iː/ (no "e")
- **I** = "ai" /aɪ/ (no "i")
- **O** = "ou" /oʊ/ (no "o")
- **U** = "iu" /juː/ (no "u")
</Aside>

```javascript
const natoAlphabet = {

  M: "Mike",      N: "November",  O: "Oscar",
  P: "Papa",      Q: "Quebec",    R: "Romeo",

  V: "Victor",    W: "Whiskey",   X: "X-ray",
  Y: "Yankee",    Z: "Zulu"
};
```

**Ejemplos:**

- My flight is **BA207**: "Bravo Alpha two zero seven" (Mi vuelo es BA207)
- The code is **XY3**: "X-ray Yankee three" (El código es XY3)
- The ID is **JK45**: "Juliett Kilo four five" (El ID es JK45)

<Aside type="note" title="Cuándo usar NATO">
Usa el alfabeto fonético cuando:
- Hay mucho ruido de fondo
- La conexión telefónica es mala
- Necesitas precisión absoluta (códigos, contraseñas, ID)
- Trabajas en aviación, militar, emergencias o soporte técnico
</Aside>

## Deletrear (Spelling)

Deletrear es la acción de decir una palabra letra por letra. Es muy útil cuando hay ruido de fondo, mala conexión o simplemente quieres asegurarte de que la otra persona entendió bien.

```javascript
const spell = (word) => {
  // "CODE"  → "si  - ou - di  - i"
  // "SMITH" → "es  - em - ai  - ti - eich"
  // "EMAIL" → "i   - em - ei  - ai - el"
  return word.split("").map(letter => names[letter]).join(" - ");
}
```

<Aside type="tip" title="Frases para pedir o dar un deletreo">
Cuando necesites deletrear algo, usa:

- "Let me spell that for you: ..." (Déjame deletrearlo: ...)
- "That's spelled ..." (Se deletrea ...)
- "Could you spell that, please?" (¿Podría deletrearlo, por favor?)
</Aside>

## Errores comunes

<Aside type="danger" title="Decir 'be' en lugar de 'bi' para la letra B">
En español la letra B se llama "be", pero en inglés se pronuncia "bi". Es el error más frecuente al deletrear y puede hacer que la otra persona escriba una letra diferente.

- **"be - ou - ou - kei"** ⛔
- **"bi - ou - ou - kei"** (book — libro) ✅
</Aside>

<Aside type="danger" title="Pronunciar H como 'ache' en lugar de 'eich'">
La letra H en inglés se pronuncia "eich", no "ache" como en español. Pronunciarla mal al deletrear produce confusión con otras letras.

- **"ache - ou - em - i"** ⛔
- **"eich - ou - em - i"** (home — hogar) ✅
</Aside>

<Aside type="danger" title="Confundir los nombres de E e I">
En español, E suena "e" e I suena "i". En inglés es al revés: E se llama "i" y la I se llama "ai". Confundirlas cambia por completo la pronunciación de una palabra deletreada.

- **"e - em - e - i - el"** ⛔
- **"i - em - ei - ai - el"** (email — correo electrónico) ✅
</Aside>
