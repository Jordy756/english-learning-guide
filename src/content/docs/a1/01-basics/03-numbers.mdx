## Números del 0 al 20

Estos son los números base que debes memorizar completamente, ya que forman la base de todos los demás:

```javascript
const basicNumbers = {
  0: "zero",      1: "one",       2: "two",
  3: "three",     4: "four",      5: "five",
  6: "six",       7: "seven",     8: "eight",
  9: "nine",      10: "ten",      11: "eleven",
  12: "twelve",   13: "thirteen", 14: "fourteen",
  15: "fifteen",  16: "sixteen",  17: "seventeen",
  18: "eighteen", 19: "nineteen", 20: "twenty"
};
```

<Aside type="tip" title="Patrón 13-19">
Del 13 al 19, el patrón general es: **número base + "teen"**
- Three → Thirteen
- Four → Fourteen
- Five → Fifteen (cambia a "fif")
- Six → Sixteen

Excepciones: eleven, twelve (no siguen ningún patrón)
</Aside>

## Decenas y Números Compuestos

Las decenas siguen un patrón consistente, y para números compuestos (21-99) usas guion:

<Tabs>
  <TabItem label="Decenas (10-90)">

```javascript
const tens = {
  10: "ten",
  20: "twenty",
  30: "thirty",
  40: "forty",
  50: "fifty",
  60: "sixty",
  70: "seventy",
  80: "eighty",
  90: "ninety"
};
```
  </TabItem>

  <TabItem label="Compuestos (21-99)">

Para números entre las decenas, une la decena y la unidad con un guion:

```javascript
const compositeNumbers = {
  21: "twenty-one",   // veintiuno
  47: "forty-seven",  // cuarenta y siete
  63: "sixty-three",  // sesenta y tres
  89: "eighty-nine"   // ochenta y nueve
};
```

  **Ejemplos:**

  - 21: twenty-one (veintiuno)
  - 47: forty-seven (cuarenta y siete)
  - 63: sixty-three (sesenta y tres)
  - 89: eighty-nine (ochenta y nueve)

  </TabItem>
</Tabs>

<Aside type="caution" title="El número 40 se escribe sin u">
El número 40 se escribe **forty** (sin "u"). Es la única decena que pierde la "u" del número base "four".
</Aside>

## Números Grandes

Para números grandes, se usan "hundred", "thousand", "million" sin plural cuando van precedidos de un número:

```javascript
const largeNumbers = {
  100:      "one hundred",   // cien
  200:      "two hundred",   // doscientos
  1000:     "one thousand",  // mil
  5000:     "five thousand", // cinco mil
  1000000:  "one million",   // un millón
  10000000: "ten million"    // diez millones
};
// Para compuestos, usa "and" después de hundred: "three hundred and forty-five"
```

**Ejemplos:**

- **456:** four hundred and fifty-six (cuatrocientos cincuenta y seis)
- **1,234:** one thousand, two hundred and thirty-four (mil doscientos treinta y cuatro)
- **50,000:** fifty thousand (cincuenta mil)
- **1,000,000:** one million (un millón)

<Aside type="note" title="Diferencia UK vs US">
En inglés británico, se usa "and" después de "hundred":  
- UK: "two hundred **and** fifty"
- US: "two hundred fifty" (sin "and")

Ambas formas son correctas, pero la versión británica es más común globalmente.
</Aside>

## Números Ordinales

Los ordinales indican posición u orden. La mayoría añaden "th", pero los tres primeros son irregulares:



```javascript
const ordinals = {
  1: "first",     // 1st - Primero
  2: "second",    // 2nd - Segundo
  3: "third",     // 3rd - Tercero
  4: "fourth",    // 4th - Cuarto
  5: "fifth",     // 5th - Quinto
  6: "sixth",     // 6th - Sexto
  7: "seventh",   // 7th - Séptimo
  8: "eighth",    // 8th - Octavo
  9: "ninth",     // 9th - Noveno
  10: "tenth"     // 10th - Décimo
};
```

**Ejemplos:**

- 21st: twenty-first (vigésimo primero)
- 32nd: thirty-second (trigésimo segundo)
- 43rd: forty-third (cuadragésimo tercero)
- 54th: fifty-fourth (quincuagésimo cuarto)

## Operaciones Matemáticas

```javascript
const mathOperations = {
  addition: "5 + 3 = 8",       // five plus three equals eight (cinco más tres es igual a ocho)
  subtraction: "10 - 4 = 6",   // ten minus four equals six (diez menos cuatro es igual a seis)
  multiplication: "3 × 2 = 6", // three times two equals six (tres por dos es igual a seis)
  division: "8 ÷ 2 = 4"        // eight divided by two equals four (ocho dividido por dos es igual a cuatro)
};
```

**Vocabulario matemático:**
- **+** : plus (más)
- **-** : minus (menos)
- **×** : times / multiplied by (por / multiplicado por)
- **÷** : divided by (dividido por)
- **=** : equals / is (es igual a / es)

## Fracciones

Las fracciones usan ordinales para el denominador:

<Tabs>
  <TabItem label="Fracciones Comunes">

```javascript
const commonFractions = {
  "1/2": "one half / a half",         // un medio (excepción)
  "1/4": "one quarter / a quarter",   // un cuarto (excepción)
  "3/4": "three quarters",            // tres cuartos
  "1/3": "one third",                 // un tercio
  "2/3": "two thirds",                // dos tercios
  "1/5": "one fifth",                 // un quinto
  "2/5": "two fifths"                 // dos quintos
};
```

  </TabItem>

  <TabItem label="Patrón General">

El denominador usa el **ordinal** de ese número. Si el numerador es mayor que 1, el ordinal lleva **"s"** al final:

```javascript
const makeFraction = (numerator, denominator) => {
  const ordinal = makeOrdinal(denominator);
  const plural = numerator > 1 ? ordinal + "s" : ordinal;
  return `${numerator} ${plural}`;
};

// Ejemplos
console.log(makeFraction(1, "eighth")); // "1 eighth"
console.log(makeFraction(3, "eighth")); // "3 eighths"
console.log(makeFraction(5, "sixth"));  // "5 sixths"
```

  **Ejemplos:**

  - 1/3: one third (un tercio)
  - 2/3: two thirds (dos tercios)
  - 3/8: three eighths (tres octavos)
  - 5/6: five sixths (cinco sextos)

  </TabItem>
</Tabs>

## Errores comunes

<Aside type="danger" title="Escribir 'fourty' en lugar de 'forty'">
El número 40 es la única decena que pierde la "u" del número base "four". No sigue el patrón de las demás.

- I have **fourty** dollars ⛔
- I have **forty** dollars (tengo cuarenta dólares) ✅
</Aside>

<Aside type="danger" title="Pluralizar hundred, thousand o million">
Cuando van precedidos de un número específico, estas palabras no se pluralizan. Solo llevan "s" en expresiones indefinidas como "hundreds of people".

- Two **hundreds** and five **thousands** ⛔
- Two **hundred** and five **thousand** (doscientos cinco mil) ✅
</Aside>

<Aside type="danger" title="Confundir los números con -teen y los de -ty">
La diferencia entre "-teen" y "-ty" está en el acento: "-teen" se acentúa al final y "-ty" al principio. Pronunciarlos igual puede causar malentendidos graves.

- She is **fifty** years old (queriendo decir quince) ⛔
- She is **fifteen** years old (ella tiene quince años) ✅
</Aside>
