Las **expresiones fijas** en inglés son como constantes predefinidas en programación: no puedes modificar su estructura interna ni traducirlas palabra por palabra. Debes memorizarlas como **bloques completos** porque su significado no se deriva de la suma de sus partes, y son fundamentales para comunicarte de manera natural y fluida en cualquier contexto.

---

## Respuestas Esenciales

Estas son expresiones que necesitarás usar constantemente en conversaciones básicas. Memorízalas como bloques completos.

<Tabs>
  <TabItem label="Confirmación">

```javascript
const confirmationResponses = {
  affirmative: [
    "Of course",        // Por supuesto
    "Sure",             // Claro
    "I see"             // Ya veo / Entiendo
  ],
  understanding: [
    "I understand",     // Entiendo
    "Got it",           // Entendido (informal)
    "That makes sense"  // Tiene sentido
  ]
};
```

**Ejemplos:**
- "**Of course** I can help you" (¡Por supuesto que puedo ayudarte!)
- "**I see** what you mean" (Ya veo lo que quieres decir)
- "**Got it**, I understand" (Entendido, comprendo)
- "**Sure**, no problem" (Claro, no hay problema)
- "**That makes sense** to me" (Eso tiene sentido para mí)

</TabItem>

  <TabItem label="Negación Suave">

```javascript
const softNegatives = {
  polite: [

    "Not quite",        // No del todo
    "I don't think so"  // No creo
  ],
  uncertain: [
    "I don't know",     // No sé
    "I'm not sure",     // No estoy seguro/a
    "Maybe"             // Quizás
  ]
};
```

**Ejemplos:**
- "**Not really**, I prefer tea" (La verdad no, prefiero té)
- "**Not quite**, it's almost correct" (No del todo, está casi correcto)
- "**I don't know** the answer" (No sé la respuesta)
- "**I'm not sure** about that" (No estoy seguro de eso)
- "**Maybe** tomorrow" (Quizás mañana)

## Fórmulas de Cortesía

Las fórmulas de cortesía son expresiones básicas que **debes usar diariamente**: **please** (por favor), **thank you** (gracias), **you're welcome** (de nada), **sorry** (perdón), e **excuse me** (disculpe). Usarlas correctamente demuestra respeto y educación en cualquier contexto.

**Ejemplos:**
- "**Please** help me" (Por favor ayúdame)
- "**Thank you** for your help" (Gracias por tu ayuda)
- "**You're welcome**, anytime" (De nada, cuando quieras)
- "**Sorry**, I didn't see you" (Perdón, no te vi)
- "**I'm sorry** for the confusion" (Lo siento por la confusión)
- "**Excuse me**, where is the exit?" (Disculpe, ¿dónde está la salida?)

<Aside type="note" title="Sorry vs Excuse me">
**Sorry** es para disculparse mientras que **Excuse me** es para llamar atención o pedir permiso
</Aside>

## Acuerdo y Desacuerdo

Estas expresiones te permiten mostrar que estás de acuerdo o en desacuerdo con alguien de manera natural.

<Tabs>
  <TabItem label="Acuerdo">

```javascript
const agreementExpressions = {
  positive: [
    "Me too",       // Yo también (con afirmativas)
    "That's right", // Es cierto / Correcto
    "I agree",      // Estoy de acuerdo
    "I think so"    // Creo que sí
  ],
  negative: [
    "Me neither",   // Yo tampoco (con negativas)
    "Neither do I"  // Yo tampoco (más formal)
  ]
};
```

**Ejemplos:**
- "I love music" → "**Me too**" (A mí también)
- "I don't like coffee" → "**Me neither**" (A mí tampoco)
- "**That's right**, the answer is correct" (Es correcto, la respuesta es correcta)
- "**I agree** with you" (Estoy de acuerdo contigo)
- "**I think so**, but I'm not certain" (Creo que sí, pero no estoy seguro)

<Aside type="caution" title="Me too con afirmativas, Me neither con negativas">
Usa **Me too** solo con afirmativas y **Me neither** solo con negativas, no al revés.
</Aside>

</TabItem>

  <TabItem label="Desacuerdo">

```javascript
const disagreementExpressions = {
  polite: [
    "I don't think so",         // No creo / No lo creo así
    "I'm not sure about that",  // No estoy seguro de eso
    "Not really"                // No exactamente
  ],
  direct: [
    "I disagree",               // No estoy de acuerdo
    "That's not right",         // Eso no es correcto
    "I don't agree"             // No estoy de acuerdo
  ]
};
```

**Ejemplos:**
- "**I don't think so**, it's unlikely" (No lo creo, es poco probable)
- "**I'm not sure about that** idea" (No estoy seguro de esa idea)
- "**Not really**, I prefer the other one" (No exactamente, prefiero el otro)
- "**I disagree** with that opinion" (No estoy de acuerdo con esa opinión)
- "**That's not right**, check again" (Eso no es correcto, verifica de nuevo)

</TabItem>
</Tabs>

## Errores Comunes

<Aside type="danger" title="Usar 'me too' con frases negativas">
- I don't like coffee → Me too ⛔
- I don't like coffee → **Me neither** (A mí tampoco) ✅
- I'm not hungry → Me too ⛔
- I'm not hungry → **Me neither** (A mí tampoco) ✅

Con negativas (not/don't), usa **me neither**, no "me too".
</Aside>

<Aside type="danger" title="Construir 'I don't know' palabra por palabra">
- I not know ⛔
- I no know ⛔
- **I don't know** (No sé) ✅

Memoriza esta expresión completa con el auxiliar "don't", no la construyas literalmente.
</Aside>

<Aside type="danger" title="Traducir 'no pasa nada' palabra por palabra">
- No passes nothing ⛔
- Nothing passes ⛔
- **No problem** (No hay problema) ✅
- **Never mind** (No importa) ✅
- **Don't worry** (No te preocupes) ✅

Memoriza estas expresiones como bloques fijos, no las traduzcas literalmente.
</Aside>

<Aside type="danger" title="Usar 'Sorry' y 'Excuse me' incorrectamente">
- Excuse me I'm late (para disculparse) ⛔
- **Sorry** I'm late (Perdón, llego tarde) ✅
- Sorry, where is the bathroom? (para llamar atención) ⛔
- **Excuse me**, where is the bathroom? (Disculpe, ¿dónde está el baño?) ✅

**Sorry** es para disculparse mientras que **Excuse me** es para llamar atención o pedir permiso
</Aside>

## Práctica

<Aside type="tip" title="Ejercicio diario">
- Grábate diciendo cada expresión en voz alta al menos 5 veces durante una semana, escucha tus grabaciones para verificar pronunciación
- Antes de dormir, imagina 3 situaciones donde usarías estas expresiones y repite en voz alta la expresión correcta
- Crea tarjetas en tu teléfono con la situación en un lado y la expresión en el otro, repasa 10 tarjetas cada mañana
- Practica frente a un espejo respondiendo a preguntas imaginarias usando estas expresiones para automatizar las respuestas
- Cuando pienses en español "no sé", oblígate mentalmente a decir "I don't know" inmediatamente, haz esto con todas las expresiones hasta que sean automáticas
</Aside>
