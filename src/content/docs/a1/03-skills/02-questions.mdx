En español, puedes convertir una afirmación en pregunta solo con entonación: "¿Tienes hambre?". En inglés, necesitas **reorganizar** la oración. Piensa en ello como una función que transforma la estructura: el verbo auxiliar se mueve al inicio.

Dominar las preguntas es esencial porque son la base de cualquier conversación: pedir información, confirmar datos, y conectar con otros.

---

## Preguntas Yes/No

Estas preguntas esperan "sí" o "no" como respuesta. La estructura depende del tipo de verbo:

<Tabs>
  <TabItem label="Con 'Be'">
    ```javascript
    const withBe = {
      structure: "Be + Subject + Complement?",
      transform: {
        statement: "She is a doctor",
        question: "Is she a doctor?"
        // ¿Es ella doctora?
      }
    };
    ```
    
    **Ejemplos:**
    - Is he your friend? (¿Es él tu amigo?)
    - Are they at home? (¿Están en casa?)
    - Was the movie good? (¿Estuvo buena la película?)
    - Were you tired? (¿Estabas cansado?)
  </TabItem>
  
  <TabItem label="Con auxiliar">
    ```javascript
    const withAuxiliary = {
      structure: "Auxiliary + Subject + Verb?",
      transform: {
        statement: "She can swim",
        question: "Can she swim?"
        // ¿Puede ella nadar?
      }
    };
    // Auxiliares: can, will, would, should, must...
    ```
    
    **Ejemplos:**
    - Can you help me? (¿Puedes ayudarme?)
    - Will they come? (¿Vendrán ellos?)
    - Should I wait? (¿Debería esperar?)
    - Have you finished? (¿Has terminado?)
  </TabItem>
  
  <TabItem label="Con Do/Does/Did">
    ```javascript
    const withDo = {
      structure: "Do/Does/Did + Subject + Base Verb?",
      present: {
        I_you_we_they: "Do you like coffee?",
        he_she_it: "Does she like coffee?"
      },
      past: "Did you see that?"
    };
    // El verbo principal va en forma base
    ```
    
    **Ejemplos:**
    - Do you speak Spanish? (¿Hablas español?)
    - Does he work here? (¿Trabaja él aquí?)
    - Did they call you? (¿Te llamaron?)
    - Do we need tickets? (¿Necesitamos boletos?)
  </TabItem>
</Tabs>

<Aside type="tip">
Cuando usas **do/does/did**, el verbo principal **siempre** va en forma base: ~~Does she likes~~ → **Does she like**
</Aside>

## Preguntas con Wh-

Las **Wh-questions** piden información específica. Comienzan con una palabra interrogativa:

```javascript
const whWords = {
  What: "qué",        // What is your name?
  Who: "quién",       // Who is she?
  Where: "dónde",     // Where do you live?
  When: "cuándo",     // When did you arrive?
  Why: "por qué",     // Why are you sad?
  Which: "cuál",      // Which one do you want?
  How: "cómo"         // How are you?
};
```

<Tabs>
  <TabItem label="Estructura">
    ```javascript
    const whStructure = {
      withBe: "Wh- + Be + Subject?",
      withAux: "Wh- + Auxiliary + Subject + Verb?",
      withDo: "Wh- + Do/Does/Did + Subject + Verb?"
    };
    ```
  </TabItem>
  
  <TabItem label="Ejemplos">
    ```javascript
    const examples = {
      withBe: [
        "Where is the station?",
        "What is your job?"
      ],
      withDo: [
        "What do you want?",
        "Where did she go?"
      ],
      withAux: [
        "When will you arrive?",
        "How can I help?"
      ]
    };
    ```
  </TabItem>
</Tabs>

**Más ejemplos:**
- What do you do? (¿A qué te dedicas?)
- Where does she live? (¿Dónde vive ella?)
- When did the movie start? (¿Cuándo empezó la película?)
- Why are you laughing? (¿Por qué te ríes?)
- How did you learn English? (¿Cómo aprendiste inglés?)

## Casos Especiales con Who/What (Sujeto)

Cuando **who** o **what** es el **sujeto** de la pregunta, **no** usamos auxiliar:

<Tabs>
  <TabItem label="Sujeto">
    ```javascript
    const subjectQuestion = {
      rule: "Who/What + Verb (conjugated)",
      // Who/What ES el sujeto
      examples: [
        "Who called you?",        // ¿Quién te llamó?
        "What happened?",         // ¿Qué pasó?
        "Who lives here?"         // ¿Quién vive aquí?
      ]
    };
    // NO: Who did call you?
    ```
  </TabItem>
  
  <TabItem label="Objeto">
    ```javascript
    const objectQuestion = {
      rule: "Who/What + Auxiliary + Subject + Verb",
      // Who/What es el OBJETO
      examples: [
        "Who did you call?",      // ¿A quién llamaste?
        "What did you buy?",      // ¿Qué compraste?
        "Who do you love?"        // ¿A quién amas?
      ]
    };
    // Aquí SÍ necesitas el auxiliar
    ```
  </TabItem>
</Tabs>

**Comparación:**
- Who broke the window? (¿Quién rompió la ventana?) — Who = sujeto
- Who did you see? (¿A quién viste?) — Who = objeto

<Aside type="caution">
Si puedes responder con un nombre en posición de sujeto ("John broke it"), la pregunta no lleva auxiliar. Si el nombre sería el objeto ("I saw John"), sí lleva auxiliar.
</Aside>

## How + Adjetivo/Adverbio

**How** se combina con adjetivos y adverbios para preguntas específicas:

```javascript
const howCombinations = {
  "How old": "¿Cuántos años?",      // How old are you?
  "How long": "¿Cuánto tiempo?",    // How long is the movie?
  "How far": "¿Qué tan lejos?",     // How far is the airport?
  "How often": "¿Qué tan seguido?", // How often do you exercise?
  "How much": "¿Cuánto? (no contable)", // How much does it cost?
  "How many": "¿Cuántos? (contable)"    // How many brothers do you have?
};
```

**Ejemplos:**
- How old is your car? (¿Cuántos años tiene tu carro?)
- How long did you wait? (¿Cuánto tiempo esperaste?)
- How often do you travel? (¿Qué tan seguido viajas?)
- How much water do you need? (¿Cuánta agua necesitas?)
- How many languages do you speak? (¿Cuántos idiomas hablas?)

## Question Tags

Los **question tags** son mini-preguntas al final para confirmar información:

<Tabs>
  <TabItem label="Regla">
    ```javascript
    const tagRule = {
      positive: "+ statement, - tag?",
      negative: "- statement, + tag?",
      // Si la oración es positiva, el tag es negativo
      // Si la oración es negativa, el tag es positivo
    };
    ```
  </TabItem>
  
  <TabItem label="Ejemplos">
    ```javascript
    const tags = {
      positive_negative: [
        "You're coming, aren't you?",
        "She can swim, can't she?",
        "They left, didn't they?"
      ],
      negative_positive: [
        "You aren't tired, are you?",
        "He doesn't smoke, does he?",
        "They weren't late, were they?"
      ]
    };
    ```
  </TabItem>
</Tabs>

**Ejemplos:**
- It's beautiful, isn't it? (Es hermoso, ¿no?)
- You like coffee, don't you? (Te gusta el café, ¿verdad?)
- She didn't call, did she? (Ella no llamó, ¿o sí?)
- We should go, shouldn't we? (Deberíamos ir, ¿no?)

## Preguntas Indirectas

Las preguntas indirectas son más formales y corteses. El orden de palabras cambia:

```javascript
const indirect = {
  direct: "Where is the station?",
  indirect: "Could you tell me where the station is?",
  // El orden vuelve a ser sujeto + verbo
  rule: "Polite phrase + Wh- + Subject + Verb"
};
```

<Tabs>
  <TabItem label="Estructura">
    ```javascript
    const politeStarters = [
      "Could you tell me...",
      "Do you know...",
      "I wonder...",
      "Can you explain..."
    ];
    // Después: wh- + subject + verb (orden normal)
    ```
  </TabItem>
  
  <TabItem label="Comparación">
    ```javascript
    const comparison = {
      direct: [
        "What time is it?",
        "Where does she work?",
        "Why did he leave?"
      ],
      indirect: [
        "Could you tell me what time it is?",
        "Do you know where she works?",
        "I wonder why he left."
      ]
    };
    // Nota: sin signo de interrogación si empieza con "I wonder"
    ```
  </TabItem>
</Tabs>

**Ejemplos:**
- Could you tell me where the bathroom is? (¿Podría decirme dónde está el baño?)
- Do you know what time the bank opens? (¿Sabes a qué hora abre el banco?)
- I was wondering if you could help me (Me preguntaba si podrías ayudarme)

## Errores Comunes

<Aside type="danger">
**Usar do/does con el verbo "be":**
- ~~Do you are tired?~~ → **Are you tired?** (¿Estás cansado?)
- ~~Does she is a student?~~ → **Is she a student?** (¿Es ella estudiante?)
</Aside>

<Aside type="danger">
**Olvidar cambiar el verbo a forma base con do/does:**
- ~~Does she likes pizza?~~ → **Does she like pizza?** (¿Le gusta la pizza?)
- ~~Did you went there?~~ → **Did you go there?** (¿Fuiste allí?)
</Aside>

<Aside type="danger">
**Usar auxiliar cuando Who/What es sujeto:**
- ~~Who did call you?~~ → **Who called you?** (¿Quién te llamó?)
- ~~What did happen?~~ → **What happened?** (¿Qué pasó?)
</Aside>

<Aside type="danger">
**Orden incorrecto en preguntas indirectas:**
- ~~Could you tell me where is the bank?~~ → **Could you tell me where the bank is?** (¿Podría decirme dónde está el banco?)
</Aside>
