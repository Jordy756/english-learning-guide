Los **linking words** (conectores o palabras de enlace) son el "pegamento" que une tus ideas en inglés. Sin ellos, tus oraciones suenan desconectadas y abruptas. Con ellos, tu discurso fluye naturalmente y suena más profesional y sofisticado. Son especialmente importantes en escritura formal, presentaciones y conversaciones de negocios.

## Categorías de Conectores

<Tabs>
  <TabItem label="Adición">

Estos conectores añaden información que **apoya** o **complementa** tu idea anterior. Son como hacer `push()` a un array de argumentos.

```javascript
const additionWords = {
  purpose: "Añadir información",
  words: [
    { word: "moreover", meaning: "además / es más" },
    { word: "furthermore", meaning: "además / asimismo" },
    { word: "in addition", meaning: "adicionalmente / además" },
    { word: "besides", meaning: "además / aparte de" },
    { word: "also", meaning: "también / además" },
  ],
  examples: [
    {
      en: "She is not only intelligent but also hardworking.",
      es: "Ella no solo es inteligente sino también trabajadora.",
    },
    {
      en: "In addition to his job, he volunteers at the local shelter.",
      es: "Además de su trabajo, él es voluntario en el refugio local.",
    },
    {
      en: "The new software is faster. Moreover, it has better security features.",
      es: "El nuevo software es más rápido. Además, tiene mejores características de seguridad.",
    }
  ],
};
```

<Aside type="tip" title="Nivel de formalidad">
- **Moreover/Furthermore:** Muy formales, ideales para escritura académica o profesional
- **In addition:** Neutral, apropiado para cualquier contexto
- **Besides:** Más casual, común en conversaciones informales
- **Also:** El más común y versátil, úsalo cuando tengas dudas
</Aside>

  </TabItem>

  <TabItem label="Contraste">

Estos conectores introducen ideas **opuestas** o **inesperadas**. Es como un `else` en tu lógica de argumentación.

```javascript
const contrastWords = {
  purpose: "Contrastar ideas",
  words: [
    { word: "however", meaning: "sin embargo / no obstante" },
    { word: "nevertheless", meaning: "no obstante / a pesar de todo" },
    { word: "on the other hand", meaning: "por otro lado / por otra parte" },
    { word: "yet", meaning: "pero / sin embargo / aún así" },
    { word: "although", meaning: "aunque / si bien / a pesar de que" },
  ],
  examples: [
    {
      en: "The project was challenging; however, we completed it on time.",
      es: "El proyecto fue desafiante; sin embargo, lo completamos a tiempo.",
    },
    {
      en: "She is very experienced. Nevertheless, she is open to learning new skills.",
      es: "Ella tiene mucha experiencia. No obstante, está abierta a aprender nuevas habilidades.",
    },
    {
      en: "I wanted to attend the meeting; on the other hand, I had a prior commitment.",
      es: "Quería asistir a la reunión; por otro lado, tenía un compromiso previo.",
    }
  ],
};
```

<Aside title="Puntuación importante">
**However** y **Nevertheless** van después de punto (.) o punto y coma (;), seguidos de coma (,):
- ✓ "He studied hard. However, he failed."
- ✓ "He studied hard; however, he failed."
- ✗ "He studied hard, however he failed."

**Although** y **Yet** pueden ir al inicio o en medio de la oración con estructuras diferentes.
</Aside>

  </TabItem>

  <TabItem label="Causa/Efecto">

Estos conectores muestran **relaciones de causa-efecto**. Es como encadenar `.then()` en promesas.

```javascript
const causeWords = {
  purpose: "Indicar causa/efecto",
  words: [
    { word: "therefore", meaning: "por lo tanto / por consiguiente" },
    { word: "thus", meaning: "así / por lo tanto / de este modo" },
    { word: "consequently", meaning: "en consecuencia / consecuentemente" },
    { word: "as a result", meaning: "como resultado / por ende" },
    { word: "because", meaning: "porque / debido a" },
    { word: "since", meaning: "ya que / dado que" },
  ],
  examples: [
    {
      en: "The server was down; therefore, the website was inaccessible.",
      es: "El servidor estaba caído; por lo tanto, el sitio web era inaccesible.",
    },
    {
      en: "She didn't meet the deadline; thus, the project was delayed.",
      es: "Ella no cumplió con la fecha límite; así, el proyecto se retrasó.",
    },
    {
      en: "He was the top performer; consequently, he received a promotion.",
      es: "Él fue el mejor desempeño; en consecuencia, recibió una promoción.",
    }
  ],
};
```

<Aside type="caution" title="Diferencia importante">
- **Therefore/Thus/Consequently/As a result:** Introducen el **efecto** o **conclusión**. Van después de la causa.
- **Because/Since:** Introducen la **razón** o **causa**. Explican el porqué.

Ejemplo: 
- "It rained. Therefore, the game was canceled." (efecto)
- "The game was canceled because it rained." (razón)
</Aside>

  </TabItem>

  <TabItem label="Secuencia">

Estos conectores organizan eventos en **orden cronológico**. Son como los pasos de un algoritmo o flujo de trabajo.

```javascript
const sequenceWords = {
  purpose: "Ordenar eventos",
  words: [
    { word: "first", meaning: "primero / en primer lugar" },
    { word: "then", meaning: "luego / entonces / después" },
    { word: "next", meaning: "siguiente / luego / a continuación" },
    { word: "finally", meaning: "finalmente / por último" },
    { word: "meanwhile", meaning: "mientras tanto / entre tanto" },
  ],
  examples: [
    {
      en: "First, we gather requirements. Then, we design the system.",
      es: "Primero, recopilamos los requisitos. Luego, diseñamos el sistema.",
    },
    {
      en: "Next, we implement the code. Finally, we test the application.",
      es: "Luego, implementamos el código. Finalmente, probamos la aplicación.",
    },
    {
      en: "I was working on the report. Meanwhile, my colleague was preparing the presentation.",
      es: "Yo estaba trabajando en el informe. Mientras tanto, mi colega estaba preparando la presentación.",
    }
  ],
};
```

<Aside type="tip" title="Uso de 'Meanwhile'">
**Meanwhile** (mientras tanto) es especial porque indica una acción **paralela** que ocurre al mismo tiempo que las otras. Es útil para describir procesos donde múltiples cosas suceden simultáneamente.

Ejemplo: "I was cooking dinner. Meanwhile, my partner was setting the table."
</Aside>

  </TabItem>

  <TabItem label="Énfasis">

Estos conectores **refuerzan** tu punto o añaden evidencia contundente. Son como añadir `console.assert()` a tu argumento.

```javascript
const emphasisWords = {
  purpose: "Enfatizar puntos",
  words: [
    { word: "indeed", meaning: "de hecho / en efecto / ciertamente" },
    { word: "in fact", meaning: "de hecho / en realidad" },
    { word: "certainly", meaning: "ciertamente / sin duda" },
    { word: "obviously", meaning: "obviamente / claramente / por supuesto" },
    { word: "clearly", meaning: "claramente / evidentemente" },
  ],
  examples: [
    {
      en: "The results are impressive. Indeed, they exceeded all expectations.",
      es: "Los resultados son impresionantes. De hecho, superaron todas las expectativas.",
    },
    {
      en: "He is an expert in his field. In fact, he has published several papers.",
      es: "Él es un experto en su campo. De hecho, ha publicado varios artículos.",
    },
    {
      en: "Obviously, we need to address this issue immediately.",
      es: "Obviamente, necesitamos abordar este problema de inmediato.",
    }
  ],
};
```

<Aside type="caution" title="Cuidado con el tono">
**Obviously** y **Clearly** pueden sonar condescendientes o arrogantes si se usan incorrectamente, como si dijeras "es obvio, ¿no lo ves?". 

Úsalos solo cuando la evidencia sea realmente evidente para todos, no para menospreciar a tu audiencia. En caso de duda, usa **Indeed** o **In fact** que son más neutrales.
</Aside>
