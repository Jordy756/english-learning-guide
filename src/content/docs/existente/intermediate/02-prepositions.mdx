Las preposiciones `in`, `on` y `at` se utilizan para indicar **tiempo** y **lugar**. La regla de oro es: **"In"** es para lo más general o grande, **"On"** es para superficies o puntos intermedios, y **"At"** es para lo más específico o pequeño. Una vez que entiendes esa lógica, elegir la preposición correcta se vuelve mucho más sencillo.

## Tabla de Preposiciones

```javascript
const Prepositions = {
  in: {
    time: "General: siglos, años, meses, estaciones, partes del día",
    place: "Espacios cerrados o áreas grandes: Países, ciudades, dentro de algo"
  },
  on: {
    time: "Más específico: Días de la semana, fechas exactas, días especiales",
    place: "Superficies o medios de transporte: Calles, objetos sobre otros, transporte público"
  },
  at: {
    time: "Muy específico: Horas precisas, momentos puntuales",
    place: "Puntos específicos: Direcciones exactas, sitios concretos de reunión"
  }
};
```

## Preposiciones de Tiempo

<Tabs>
  <TabItem label="In (General)">

```javascript
// IN: Para periodos generales de tiempo
const timeExamples = {
  centuries: "in the 21st century",
  years: "in 2024",
  months: "in January",
  seasons: "in summer",
  timeOfDay: "in the morning / in the afternoon"
};

console.log(`I was born ${timeExamples.years}`);      // "I was born in 2024"
console.log(`We travel ${timeExamples.seasons}`);     // "We travel in summer"
console.log(`I exercise ${timeExamples.timeOfDay}`);  // "I exercise in the morning"
```

  </TabItem>

  <TabItem label="On (Específico)">

```javascript
// ON: Para días y fechas específicas
const timeExamples = {
  weekDays: "on Monday",
  dates: "on December 25th",
  specialDays: "on my birthday"
};

console.log(`I work ${timeExamples.weekDays}`);      // "I work on Monday"
console.log(`We celebrate ${timeExamples.dates}`);   // "We celebrate on December 25th"
console.log(`I party ${timeExamples.specialDays}`);  // "I party on my birthday"
```

  </TabItem>

  <TabItem label="At (Muy Específico)">

```javascript
// AT: Para horas y momentos puntuales
const timeExamples = {
  hours: "at 3:00 PM",
  moments: "at noon / at midnight",
  night: "at night"  // Excepción importante
};

console.log(`The meeting is ${timeExamples.hours}`);    // "The meeting is at 3:00 PM"
console.log(`I eat lunch ${timeExamples.moments}`);     // "I eat lunch at noon"
console.log(`I study ${timeExamples.night}`);           // "I study at night"
```

  </TabItem>
</Tabs>

:::caution[Excepción de Tiempo]
Decimos "In the morning", "In the afternoon", pero para la noche siempre es **"At night"**.
:::

## Preposiciones de Lugar

<Tabs>
  <TabItem label="In (Áreas Grandes)">

```javascript
// IN: Para espacios cerrados o áreas grandes
const placeExamples = {
  countries: "in Mexico",
  cities: "in New York",
  inside: "in the car / in the room"
};

console.log(`I live ${placeExamples.countries}`);   // "I live in Mexico"
console.log(`She works ${placeExamples.cities}`);   // "She works in New York"
console.log(`The keys are ${placeExamples.inside}`);// "The keys are in the car"
```

  </TabItem>

  <TabItem label="On (Superficies)">

```javascript
// ON: Para superficies y transporte grande
const placeExamples = {
  streets: "on Main Street",
  surfaces: "on the table",
  transport: "on the bus / on the plane"
};

console.log(`I live ${placeExamples.streets}`);      // "I live on Main Street"
console.log(`The book is ${placeExamples.surfaces}`);// "The book is on the table"
console.log(`She is ${placeExamples.transport}`);    // "She is on the bus"
```

  </TabItem>

  <TabItem label="At (Puntos Específicos)">

```javascript
// AT: Para direcciones exactas y puntos de reunión
const placeExamples = {
  addresses: "at 123 Main Street",
  places: "at the office / at school",
  meetingPoints: "at the bus stop"
};

console.log(`I am ${placeExamples.addresses}`);      // "I am at 123 Main Street"
console.log(`She is ${placeExamples.places}`);       // "She is at the office"
console.log(`Meet me ${placeExamples.meetingPoints}`);// "Meet me at the bus stop"
```

  </TabItem>
</Tabs>

## Nota sobre Transporte


  <Card title="In (Vehículos Pequeños)" icon="approve-check">
    Usas **"In"** para vehículos pequeños donde no puedes caminar.
    ```javascript
    const smallVehicles = ["in a car", "in a taxi"];
    ```
  </Card>
  <Card title="On (Vehículos Grandes)" icon="rocket">
    Usas **"On"** para vehículos grandes donde puedes estar de pie o caminar.
    ```javascript
    const largeVehicles = ["on a plane", "on a train", "on a ship"];
    ```
  </Card>
