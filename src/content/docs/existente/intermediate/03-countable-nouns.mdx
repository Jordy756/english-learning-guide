En inglés, los sustantivos se dividen en dos categorías: **contables** (countable) e **incontables** (uncountable). Esta distinción afecta cómo usamos artículos, cuantificadores y verbos.

## ¿Cuál es la diferencia?

```javascript
const NounTypes = {
  countable: {
    definition: "Se pueden contar individualmente",
    singular: true,
    plural: true,
    examples: ["apple", "car", "idea", "book"]
  },
  uncountable: {
    definition: "No se pueden contar individualmente",
    singular: true,
    plural: false,  // No tienen forma plural
    examples: ["water", "music", "information", "money"]
  }
};
```

```javascript
// COUNTABLE: Tienen singular y plural
const countableNouns = {

```

  </TabItem>

  <TabItem label="Incontables">

```javascript
// UNCOUNTABLE: Solo forma singular, no se pueden contar
const uncountableNouns = {
  "water":       "agua",
  "music":       "música",
  "information": "información",
  "money":       "dinero",
  "advice":      "consejo(s)",
  "furniture":   "muebles",
  "bread":       "pan",
  "rice":        "arroz"
};

// Ejemplos en oraciones
console.log("I need water");          // Necesito agua
console.log("She gave me advice");    // Ella me dio consejo
// ❌ I need waters (INCORRECTO)
// ❌ She gave me advices (INCORRECTO)
```

  </TabItem>
</Tabs>

:::caution[Cuidado]
Algunos sustantivos que son contables en español son **incontables** en inglés:
- **Information** (información) → ❌ informations
- **Advice** (consejo) → ❌ advices
- **Furniture** (muebles) → ❌ furnitures
- **News** (noticias) → ❌ newses
:::

## Cuantificadores

Los cuantificadores cambian dependiendo del tipo de sustantivo:

<CardGrid>
  <Card title="Solo Contables">
    - **a / an** (un/una)
    - **many** (muchos)
    - **few** (pocos)
    - **a few** (algunos)
    - **several** (varios)
  </Card>
  <Card title="Solo Incontables">
    - **much** (mucho)
    - **little** (poco)
    - **a little** (un poco)
    - **a bit of** (un poco de)
  </Card>
  <Card title="Ambos">
    - **some** (algo de / algunos)
    - **any** (algo de / alguno)
    - **a lot of** (mucho/muchos)
    - **plenty of** (bastante)
    - **no** (ningún/nada de)
  </Card>
</CardGrid>

## Ejemplos con Cuantificadores

<Tabs>
  <TabItem label="Many vs Much">

```javascript
// MANY → Contables (plural)
// MUCH → Incontables

const manyVsMuch = {
  // ✅ Correcto
  correct: [
    "How many books do you have?",      // Contable
    "How much water do you need?",      // Incontable
    "There are many people here",       // Contable
    "I don't have much time"            // Incontable
  ],
  // ❌ Incorrecto
  incorrect: [
    "How much books...",    // ❌ books es contable
    "How many water..."     // ❌ water es incontable
  ]
};
```

  </TabItem>

  <TabItem label="Few vs Little">

```javascript
// FEW → Contables (pocos)
// LITTLE → Incontables (poco)

const fewVsLittle = {
  // FEW vs A FEW
  few: "Few people came",           // Pocas personas (negativo)
  aFew: "A few people came",        // Algunas personas (positivo)
  
  // LITTLE vs A LITTLE
  little: "There is little hope",   // Hay poca esperanza (negativo)
  aLittle: "There is a little hope" // Hay algo de esperanza (positivo)
};

// La diferencia entre "few/little" y "a few/a little"
// es que "a few/a little" suena más positivo
```

  </TabItem>

  <TabItem label="Some vs Any">

```javascript
// SOME → Oraciones afirmativas y ofrecimientos
// ANY → Oraciones negativas y preguntas

const someVsAny = {
  affirmative: [
    "I have some friends",           // Tengo algunos amigos
    "There is some milk"             // Hay algo de leche
  ],
  negative: [
    "I don't have any friends",      // No tengo ningún amigo
    "There isn't any milk"           // No hay nada de leche
  ],
  questions: [
    "Do you have any questions?",    // ¿Tienes alguna pregunta?
    "Is there any coffee?"           // ¿Hay algo de café?
  ],
  offers: [
    "Would you like some coffee?",   // ¿Quieres algo de café?
    "Can I have some water?"         // ¿Puedo tener algo de agua?
  ]
};
```

  </TabItem>
</Tabs>

## Cómo contar lo incontable

Para contar sustantivos incontables, usamos **contenedores** o **unidades de medida**:

```javascript
const countingUncountable = {
  water: [
    "a glass of water",      // un vaso de agua
    "a bottle of water",     // una botella de agua
    "two liters of water"    // dos litros de agua
  ],
  bread: [
    "a slice of bread",      // una rebanada de pan
    "a loaf of bread",       // una barra de pan
    "two pieces of bread"    // dos pedazos de pan
  ],
  advice: [
    "a piece of advice",     // un consejo
    "some pieces of advice"  // algunos consejos
  ],
  information: [
    "a piece of information", // una información
    "some information"        // algo de información
  ],
  furniture: [
    "a piece of furniture",   // un mueble
    "some pieces of furniture" // algunos muebles
  ]
};
```

## Resumen Rápido

| Pregunta | Contable | Incontable |
|----------|----------|------------|
| ¿Tiene plural? | ✅ Sí | ❌ No |
| ¿Usa a/an? | ✅ Sí | ❌ No |
| ¿Cuántos? | How **many**? | How **much**? |
| ¿Poco? | **Few** / A few | **Little** / A little |
| ¿Mucho? | **Many** | **Much** |
| ¿Ambos? | A lot of, some, any | A lot of, some, any |
