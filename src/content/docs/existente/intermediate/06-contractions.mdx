Las contracciones son formas abreviadas de palabras o frases que se usan comúnmente en el inglés hablado y escrito para hacer que el lenguaje suene más natural y fluido.

## Futuro con "'ll" (Will)

Se usa para hablar de acciones que sucederán después.

```javascript
const FutureContractions = {
  "I'll":        { full: "I will",       translation: "Yo (haré/iré/seré...)" },
  "You'll":      { full: "You will",     translation: "Tú (harás/irás...)" },
  "He'll":       { full: "He will",      translation: "Él (hará...)" },
  "She'll":      { full: "She will",     translation: "Ella (hará...)" },
  "It'll":       { full: "It will",      translation: "Eso (será/hará...)" },
  "We'll":       { full: "We will",      translation: "Nosotros (haremos...)" },
  "They'll":     { full: "They will",    translation: "Ellos (harán...)" }
};
```

## El misterio de la "d" (Would vs. Had)

Puede significar dos cosas diferentes. Aquí tienes cómo distinguirlas:

<Tabs>
  <TabItem label="Would (Condicional)">

```javascript
// Cuando el verbo que sigue está en forma BASE
const WouldExamples = {
  "I'd":      "I would eat",      // Yo comería
  "You'd":    "You would play",   // Tú jugarías
  "He'd":     "He would work",    // Él trabajaría
  "She'd":    "She would help",   // Ella ayudaría
  "We'd":     "We would stay",    // Nosotros nos quedaríamos
  "They'd":   "They would go"     // Ellos irían
};
```

  </TabItem>

  <TabItem label="Had (Pasado Perfecto)">

```javascript
// Cuando el verbo que sigue es PARTICIPIO (-ed, -en, irregular)
const HadExamples = {
  "I'd":      "I had eaten",      // Yo había comido
  "You'd":    "You had played",   // Tú habías jugado
  "He'd":     "He had worked",    // Él había trabajado
  "She'd":    "She had helped",   // Ella había ayudado
  "We'd":     "We had stayed",    // Nosotros nos habíamos quedado
  "They'd":   "They had gone"     // Ellos habían ido
};
```

  </TabItem>
</Tabs>

:::note[Nota]
Para saber cuál es cuál, fíjate siempre en la palabra que sigue. Si termina en **-ed** o es un participio irregular (como *gone* o *seen*), casi siempre es **Had**.
:::

## Verbo "To Be" (Am, Is, Are)

Se usan para decir quién es alguien o dónde está.

```javascript
const ToBeContractions = {
  "I'm":        { full: "I am",       translation: "Yo soy / estoy" },
  "You're":     { full: "You are",    translation: "Tú eres / estás" },
  "He's":       { full: "He is",      translation: "Él es o está" },
  "She's":      { full: "She is",     translation: "Ella es o está" },
  "It's":       { full: "It is",      translation: "Eso es / está" },
  "We're":      { full: "We are",     translation: "Nosotros somos / estamos" },
  "They're":    { full: "They are",   translation: "Ellos son / están" }
};
```

## Verbo "Have" (Have, Has)

Se usan principalmente en tiempos compuestos (como "Yo he comido").

```javascript
const HaveContractions = {
  "I've":       { full: "I have",      translation: "Yo he / tengo" },
  "You've":     { full: "You have",    translation: "Tú has / tienes" },
  "We've":      { full: "We have",     translation: "Nosotros hemos / tenemos" },
  "They've":    { full: "They have",   translation: "Ellos han / tienen" },
  "He's":       { full: "He has",      translation: "Él ha" },
  "She's":      { full: "She has",     translation: "Ella ha" }
};
```

:::caution[Cuidado]
**He's** y **She's** pueden significar tanto "He/She is" como "He/She has". El contexto te dirá cuál es:
- **He's a doctor** = He is a doctor (Él es doctor)
- **He's eaten** = He has eaten (Él ha comido)
:::

## Negaciones con "Not" (n't)

Estas son fundamentales para negar acciones.

<CardGrid>
  <Card title="Do / Does">
    - **Don't** = Do not (para verbos normales)
    - **Doesn't** = Does not (para He/She/It)
  </Card>
  <Card title="Be (Is, Are, Was, Were)">
    - **Isn't** = Is not
    - **Aren't** = Are not
    - **Wasn't** = Was not
    - **Weren't** = Were not
  </Card>
  <Card title="Modales">
    - **Can't** = Cannot (no poder)
    - **Won't** = Will not (futuro negativo)
    - **Wouldn't** = Would not
    - **Shouldn't** = Should not
    - **Couldn't** = Could not
  </Card>
  <Card title="Have / Has">
    - **Haven't** = Have not
    - **Hasn't** = Has not
  </Card>
</CardGrid>

## Ejemplos en contexto

| Oración con contracción | Forma completa | Traducción |
|-------------------------|----------------|------------|
| I'm going to the store | I am going to the store | Voy a la tienda |
| She's been working hard | She has been working hard | Ella ha estado trabajando duro |
| We'll see you tomorrow | We will see you tomorrow | Te veremos mañana |
| They'd already left | They had already left | Ellos ya se habían ido |
| I can't believe it | I cannot believe it | No puedo creerlo |
