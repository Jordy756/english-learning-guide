Estos tres verbos son fundamentales porque cambian el **"matiz"** de lo que decimos. Todos son seguidos por un verbo en su forma base (sin "to").

## Tabla de Verbos Modales

```javascript
const ModalVerbs = {
  would: {
    translation: "-ía (terminación verbal)",
    usage: "Situaciones hipotéticas o deseos",
    example: "I would travel more"
  },
  should: {
    translation: "Debería",
    usage: "Dar consejos o sugerencias",
    example: "You should study today"
  },
  could: {
    translation: "Podría / Podía",
    usage: "Posibilidad o habilidad en el pasado",
    example: "She could help us"
  }
};
```

## Uso de Cada Verbo Modal

<Tabs>
  <TabItem label="Would">

```javascript
// WOULD: Situaciones hipotéticas o deseos
const wouldExamples = {
  hypothetical: "I would travel more if I had money",
  desire: "I would like a coffee",
  courtesy: "Would you help me?"
};

console.log(wouldExamples.hypothetical);  // "I would travel more if I had money"
console.log(wouldExamples.desire);        // "I would like a coffee"
console.log(wouldExamples.courtesy);      // "Would you help me?"
```

  </TabItem>

  <TabItem label="Should">

```javascript
// SHOULD: Dar consejos o sugerencias
const shouldExamples = {
  advice: "You should study today",
  suggestion: "She should see a doctor",
  recommendation: "We should leave early"
};

console.log(shouldExamples.advice);          // "You should study today"
console.log(shouldExamples.suggestion);      // "She should see a doctor"
console.log(shouldExamples.recommendation);  // "We should leave early"
```

  </TabItem>

  <TabItem label="Could">

```javascript
// COULD: Posibilidad o habilidad en el pasado
const couldExamples = {
  possibility: "She could help us",
  past: "I could swim when I was 5",
  request: "Could you open the window?"
};

console.log(couldExamples.possibility);  // "She could help us"
console.log(couldExamples.past);         // "I could swim when I was 5"
console.log(couldExamples.request);      // "Could you open the window?"
```

  </TabItem>

## Estructura: Afirmativo, Negativo, Pregunta

```javascript
// Los verbos modales NO cambian con la persona
// I/you/he/she/it/we/they + modal + verbo base

const buildSentence = (subject, modal, verb) => {
  return {
    affirmative: `${subject} ${modal} ${verb}`,
    negative: `${subject} ${modal} not ${verb}`,
    question: `${modal} ${subject} ${verb}?`
  };
};

const example = buildSentence("She", "should", "work today");

console.log(example.affirmative);  // "She should work today"
console.log(example.negative);     // "She should not work today"
console.log(example.question);     // "Should she work today?"
```

## Reglas Importantes

  <Card title="Forma Base" icon="approve-check">
    Después de **would**, **should** y **could** el verbo va siempre en **forma base**.
    ```javascript
    // ✅ Correcto
    "should go", "could see", "would like"
    
    // ❌ Incorrecto
    "should goes", "could seeing"
    ```
  </Card>
  <Card title="No Cambian" icon="setting">
    **No cambian** con la persona: I/you/he/she/it/we/they should (no existe "shoulds").
    ```javascript
    // ✅ Todos usan la misma forma
    "I should", "He should", "They should"
    ```
  </Card>



  <Card title="Sin 'To'" icon="error">
    Los verbos modales **no aceptan el "to"** después de ellos.
    ```javascript
    // ✅ Correcto
    "I would go"
    
    // ❌ Incorrecto
    "I would to go"
    ```
  </Card>
  <Card title="Verbo Principal" icon="information">
    En negativo o pregunta, el verbo principal se mantiene en **forma base**.
    ```javascript
    "She should not work"  // No "works"
    "Should she work?"     // No "works"
    ```
  </Card>


:::note[¿Qué es Forma Base?]
Es el verbo en su estado puro. **No** se le añade nada al final (ni -s, -es, o -ies), sin importar la persona.
:::
