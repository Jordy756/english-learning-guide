## Formas de uso

El pasado simple tiene tres formas: afirmativa, negativa e interrogativa. A diferencia del presente simple, en pasado **todos los sujetos usan la misma forma del verbo**.

### Afirmaciones

Para construir oraciones afirmativas, usa: **Sujeto + Verbo en pasado + Complemento**. 

```javascript

  eat: "ate"
};

  if (irregularPastForms[verb]) {
    return irregularPastForms[verb];
  }
  // Si es regular, aplica las reglas del -ed
  return conjugateRegular(verb);
};

  let pastVerb = conjugatePast(verb);
  
  // "Be" es especial: usa "was" con I/he/she/it, "were" con you/we/they
  if (verb === "be") {
    pastVerb = ["I", "he", "she", "it"].includes(subject) ? "was" : "were";
  }
  
  console.log(`${subject} ${pastVerb} ${complement}`);
};

// Ejemplos de uso

affirm("She", "go", "to the store");               // "She went to the store"
affirm("They", "play", "soccer last week");        // "They played soccer last week"
affirm("He", "eat", "pizza for dinner");           // "He ate pizza for dinner"
affirm("We", "be", "friends in college");          // "We were friends in college"
affirm("It", "be", "a great day");                 // "It was a great day"
```

### Negaciones

Para formar negaciones en pasado simple, usa el auxiliar **did not** (didn't) + verbo en forma base. La estructura es: **Sujeto + did not + Verbo base + Complemento**.

  if (/[^aeiou][aeiou][^aeiouwxy]$/.test(verb) && verb.length <= 4) {
    return verb + verb.slice(-1) + 'ed';
  }
  return verb + 'ed';
};

const getEDPronunciation = (verb) => {
  const lastLetter = verb.slice(-1).toLowerCase();
  const lastTwoLetters = verb.slice(-2).toLowerCase();
  
  // /…™d/ - si termina en t o d
  if (lastLetter === 't' || lastLetter === 'd') {

  }
  
  // /t/ - despu√©s de sonidos sordos
  const voiceless = ['p', 'k', 'f', 's', 'x'];
  if (voiceless.includes(lastLetter) || 
      lastTwoLetters === 'sh' || 
      lastTwoLetters === 'ch' || 
      lastTwoLetters === 'ss') {
    return {
      sound: '/t/',
      description: 'Suena como "t"',
      example: `${verb}ed suena como "${verb}t"`
    };
  }
  
  // /d/ - despu√©s de sonidos sonoros (vocales y consonantes sonoras)
  return {
    sound: '/d/',
    description: 'Suena como "d"',
    example: `${verb}ed suena como "${verb}d"`
  };
};

// FUNCI√ìN PRINCIPAL: Convierte verbo a pasado simple con pronunciaci√≥n
const convertToPast = (verb) => {
  verb = verb.toLowerCase().trim();
  
  // 1. Verificar si es irregular
  if (irregularVerbsDB[verb]) {

      verb: verb,

  // 2. Si es regular, conjugar y obtener pronunciaci√≥n

  };
};

## Errores comunes que debes evitar

<Aside type="danger" title="Bug #1: Usar el pasado con 'did'">
Cuando usas el auxiliar `did` o `didn't`, el verbo principal DEBE estar en forma base.

```javascript
// ‚ùå BUG: El verbo est√° en pasado cuando no debe
console.log("Did you went to the store?");  // INCORRECTO
console.log("She didn't wrote the email");  // INCORRECTO

// ‚úÖ FIX: Usa la forma base del verbo
console.log("Did you go to the store?");    // CORRECTO
console.log("She didn't write the email");  // CORRECTO
```
</Aside>

<Aside type="danger" title="Bug #2: Confundir 'was/were' con otros verbos">
El verbo "to be" no usa `did` para preguntas o negaciones en pasado.

```javascript
// ‚ùå BUG: "To be" no necesita auxiliar
console.log("Did you were happy?");         // INCORRECTO
console.log("He didn't was there");         // INCORRECTO

// ‚úÖ FIX: Usa "was/were" directamente
console.log("Were you happy?");             // CORRECTO
console.log("He wasn't there");             // CORRECTO
```
</Aside>

<Aside type="danger" title="Bug #3: Olvidar cambiar la forma del verbo en afirmaciones">
En oraciones afirmativas, NO uses el auxiliar `did`. El verbo va directamente en pasado.

```javascript
// ‚ùå BUG: No uses "did" en afirmaciones
console.log("I did went to school");        // INCORRECTO
console.log("She did ate breakfast");       // INCORRECTO

// ‚úÖ FIX: Solo usa el verbo en pasado
console.log("I went to school");            // CORRECTO
console.log("She ate breakfast");           // CORRECTO
```
</Aside>
