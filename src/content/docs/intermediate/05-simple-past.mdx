El pasado simple es el tiempo verbal que utilizas para hablar de acciones completadas en el pasado. Es como la función de historial en tu código: registra eventos que ya ocurrieron y terminaron en un punto específico del tiempo. Ya sea que hables de algo que pasó hace cinco minutos o hace cinco años, el pasado simple es tu herramienta principal.

## ¿Cuándo usar el pasado simple?

Este tiempo verbal se usa en tres situaciones principales:

1. **Acciones completadas en el pasado:** "I wrote the code yesterday" (Escribí el código ayer)
2. **Secuencias de acciones:** "She opened the app, logged in, and started working" (Ella abrió la app, inició sesión, y empezó a trabajar)
3. **Hábitos del pasado:** "When I was young, I played video games every day" (Cuando era joven, jugaba videojuegos todos los días)

## Tipos de verbos

En inglés, los verbos se dividen en dos categorías cuando hablamos del pasado: **regulares** e **irregulares**. La diferencia está en cómo forman su pasado simple.

### Verbos regulares

Los verbos regulares son predecibles. Simplemente añades `-ed` al final del verbo, sin importar el sujeto. 

Antes de ver la lista, es importante conocer las **reglas de ortografía** para conjugar correctamente:

1. **Termina en -e:** Solo añade `-d` (like → liked, dance → danced)
2. **Consonante + y:** Cambia la "y" por "ied" (study → studied, carry → carried)
3. **CVC (Consonante-Vocal-Consonante) en sílaba acentuada:** Duplica la última consonante y añade `-ed` (stop → stopped, plan → planned)
4. **Regla general:** Añade `-ed` (work → worked, play → played)

Ahora que conoces las reglas, aquí está la función para conjugar cualquier verbo regular automáticamente:

```javascript
const conjugateRegularVerb = (verb) => {
  // 1. Ya termina en -e: solo añade -d
  if (verb.endsWith("e")) return verb + "d";

  // 2. Termina en consonante + y: cambia y por ied
  if (/[^aeiou]y$/.test(verb)) return verb.slice(0, -1) + "ied";

  /* 3. Termina en CVC (consonante-vocal-consonante) con una sílaba
        y la última sílaba es acentuada: duplica la última consonante + ed */
  if (/[^aeiou][aeiou][^aeiouwxy]$/.test(verb) && verb.length <= 4) return verb + verb.slice(-1) + "ed";

  // 4. Regla general: añade -ed
  return verb + "ed";
};

// Ejemplos
console.log(conjugateRegularVerb("work")); // "worked" (regla general)
console.log(conjugateRegularVerb("like")); // "liked" (termina en -e)
console.log(conjugateRegularVerb("study")); // "studied" (consonante + y)
console.log(conjugateRegularVerb("stop")); // "stopped" (duplica consonante)
console.log(conjugateRegularVerb("play")); // "played" (vocal + y, regla general)
```

<Aside type="tip" title="Truco para recordar">
La **mayoría** de los verbos solo necesitan `-ed`. Las otras reglas son para hacer que la palabra se vea y suene más natural. Por ejemplo, "likeed" se vería extraño, por eso solo añadimos `-d` a verbos que ya terminan en "e".
</Aside>

## Formas de uso

El pasado simple tiene tres formas: afirmativa, negativa e interrogativa. A diferencia del presente simple, en pasado **todos los sujetos usan la misma forma del verbo**.

### Afirmaciones

Para construir oraciones afirmativas, usa: **Sujeto + Verbo en pasado + Complemento**. 

```javascript
// Base de datos de verbos irregulares (algunos ejemplos)
const irregularPastForms = {
  go: "went",
  have: "had",
  do: "did",
  see: "saw",
  make: "made",
  get: "got",
  eat: "ate",
};

const conjugatePast = (verb) => {
  const irregularVerb = irregularPastForms[verb];

  // Si es irregular, retorna su forma irregular; si no, aplica las reglas del -ed
  return irregularVerb || conjugateRegularVerb(verb);
};

const affirm = ({ subject, verb, complement }) => {
  subject = subject.trim();
  verb = verb.trim().toLowerCase();
  complement = complement.trim();

  let pastVerb = "";

  // Manejo de los verbos "am", "is", "are"
  if (["am", "is", "are"].includes(verb)) {
    // Para "am", "is", "are", el pasado es "was" o "were" dependiendo del sujeto
    pastVerb = ["I", "he", "she", "it"].includes(subject) ? "was" : "were";
  } else {
    // Para otros verbos, conjugamos al pasado
    pastVerb = conjugatePast(verb);
  }

  return `${subject} ${pastVerb} ${complement}`;
};

// Ejemplos de uso
console.log(affirm({ subject: "I", verb: "am", complement: "happy" })); 
// "I was happy" → "Yo estaba feliz"
console.log(affirm({ subject: "I", verb: "work", complement: "yesterday" })); 
// "I worked yesterday" → "Yo trabajé ayer"

console.log(affirm({ subject: "She", verb: "go", complement: "to the store" })); 
// "She went to the store" → "Ella fue a la tienda"

console.log(affirm({ subject: "They", verb: "play", complement: "soccer last week" })); 
// "They played soccer last week" → "Ellos jugaron fútbol la semana pasada"

console.log(affirm({ subject: "He", verb: "eat", complement: "pizza for dinner" })); 
// "He ate pizza for dinner" → "Él comió pizza para la cena"

console.log(affirm({ subject: "We", verb: "are", complement: "friends in college" })); 
// "We were friends in college" → "Nosotros éramos amigos en la universidad"

console.log(affirm({ subject: "It", verb: "is", complement: "a great day" })); 
// "It was a great day" → "Fue un gran día"
```

### Negaciones

Para formar negaciones en pasado simple, usa el auxiliar **did not** (didn't) + verbo en forma base. La estructura es: **Sujeto + did not + Verbo base + Complemento**.

<Aside title="Regla importante">
Cuando usas `did` o `did not`, el verbo principal **siempre** vuelve a su forma base (infinitivo sin "to"). El auxiliar `did` ya indica que estamos hablando del pasado, por eso el verbo no necesita cambiar.
</Aside>

```javascript
const deny = ({ subject, verb, complement }) => {
  verb = verb.trim().toLowerCase();

  return ["am", "is", "are"].includes(verb)
    ? `${subject} ${verb === "are" ? "were" : "was"} not ${complement}`
    : `${subject} did not ${verb} ${complement}`;
};

// Ejemplos de uso:
console.log(deny({ subject: "I", verb: "work", complement: "yesterday" }));
// "I did not work yesterday" → "No trabajé ayer"

console.log(deny({ subject: "She", verb: "go", complement: "to the party" }));
// "She did not go to the party" → "Ella no fue a la fiesta"

console.log(deny({ subject: "They", verb: "play", complement: "video games" }));
// "They did not play video games" → "Ellos no jugaron videojuegos"

console.log(deny({ subject: "He", verb: "eat", complement: "breakfast" }));
// "He did not eat breakfast" → "Él no desayunó"

console.log(deny({ subject: "We", verb: "are", complement: "late" }));
// "We were not late" → "No llegamos tarde"
```

<Aside type="caution" title="Error común">
**Incorrecto:** "I didn't went to school" ✗\
**Correcto:** "I didn't go to school" ✓

Recuerda: cuando usas `did` o `didn't`, el verbo vuelve a su forma base. No uses "went" (pasado), usa "go" (base).
</Aside>

### Preguntas

Para hacer preguntas, coloca **did** al inicio, seguido del sujeto y el verbo en forma base. La estructura es: **Did + Sujeto + Verbo base + Complemento + ?**

```javascript
const ask = ({ subject, verb, complement }) => {
  verb = verb.trim().toLowerCase();

  return ["am", "is", "are"].includes(verb)
    ? `${verb === "am" || verb === "is" ? "Was" : "Were"} ${subject} ${complement}?`
    : `Did ${subject} ${verb} ${complement}?`;
};

// Ejemplos de uso:
console.log(ask({ subject: "you", verb: "work", complement: "yesterday" }));
// "Did you work yesterday?" → "¿Trabajaste ayer?"

console.log(ask({ subject: "she", verb: "go", complement: "to the meeting" }));
// "Did she go to the meeting?" → "¿Ella fue a la reunión?"

console.log(ask({ subject: "they", verb: "play", complement: "well" }));
// "Did they play well?" → "¿Ellos jugaron bien?"

console.log(ask({ subject: "he", verb: "eat", complement: "lunch" }));
// "Did he eat lunch?" → "¿Él almorzó?"

console.log(ask({ subject: "we", verb: "are", complement: "at the park" }));
// "Were we at the park?" → "¿Estuvimos en el parque?"

console.log(ask({ subject: "I", verb: "am", complement: "late" }));
// "Was I late?" → "¿Llegué tarde?"

console.log(ask({ subject: "she", verb: "is", complement: "ready" }));
// "Was she ready?" → "¿Ella estaba lista?"
```

<Aside title='Pregunta única: "To be" en pasado'>
El verbo **to be** es especial incluso en pasado. No necesita el auxiliar `did` para preguntas o negaciones. En su lugar, inviertes el orden directamente:

**Preguntas:**
- "**Was** he at home?" ✓ (no "Did he be at home?" ✗)
- "**Were** you tired?" ✓ (no "Did you be tired?" ✗)

**Negaciones:**
- "He **was not** (wasn't) there" ✓ (no "He did not be there" ✗)
- "They **were not** (weren't) happy" ✓ (no "They did not be happy" ✗)
</Aside>

## Pronunciación del -ed en verbos regulares

Una de las partes más importantes del pasado simple es saber cómo pronunciar la terminación `-ed`. Hay tres posibles pronunciaciones: **/t/**, **/d/** o **/ɪd/** (id), y dependen del sonido final del verbo base.

<CardGrid>
  <Card title="Pronunciación /t/" icon="star">
    **Después de sonidos sordos**: p, k, f, s, sh, ch, x
    
    **Ejemplos:**
    - worked → /wɜrkt/
    - stopped → /stɑpt/
    - watched → /wɑtʃt/
    - washed → /wɑʃt/
    - laughed → /læft/
  </Card>

  <Card title="Pronunciación /ɪd/" icon="add-document">
    **Después de T o D** (añade sílaba extra)

    - needed → /ˈni·dɪd/ (2 sílabas)
    - started → /ˈstɑr·tɪd/ (2 sílabas)
    - ended → /ˈen·dɪd/ (2 sílabas)
  </Card>

  <Card title="Pronunciación /d/" icon="approve-check">
    **Después de sonidos sonoros**: vocales, b, g, v, z, m, n, l, r

## Expresiones de tiempo comunes con pasado simple

El pasado simple suele ir acompañado de expresiones de tiempo que indican cuándo ocurrió la acción:

```javascript
const timeExpressions = [
  "yesterday",                    // ayer
  "last night/week/month/year",   // anoche/la semana/mes/año pasado
  "ago" ,                         // hace (two days ago = hace dos días)
  "in 2020",                      // en 2020
  "when I was young",             // cuando era joven
  "at that time",                 // en ese momento
  "once upon a time",             // érase una vez
  "the other day"                 // el otro día
];

// Ejemplos en oraciones
const examples = [
  "I worked from home yesterday",                  // Trabajé desde casa ayer
  "She visited Paris last summer",                 // Ella visitó París el verano pasado
  "They moved to New York three years ago",        // Hace tres años se mudaron a Nueva York
  "He graduated in 2019",                          // En 2019 se graduó
  "When I was young, I played soccer every day",   // Cuando era joven, jugaba fútbol todos los días
  "We met at that time",                           // Nos encontramos en ese momento
  "Once upon a time, there was a king",            // Érase una vez, había un rey
  "I saw him the other day"                        // Lo vi el otro día
];
```

## Errores comunes que debes evitar

<Aside type="danger" title="Bug #1: Usar el pasado con 'did'">
Cuando usas el auxiliar `did` o `didn't`, el verbo principal DEBE estar en forma base.

```javascript
// ❌ BUG: El verbo está en pasado cuando no debe
console.log("Did you went to the store?");  // INCORRECTO
console.log("She didn't wrote the email");  // INCORRECTO

// ✅ FIX: Usa la forma base del verbo
console.log("Did you go to the store?");    // CORRECTO
console.log("She didn't write the email");  // CORRECTO
```
</Aside>

<Aside type="danger" title="Bug #2: Confundir 'was/were' con otros verbos">
El verbo "to be" no usa `did` para preguntas o negaciones en pasado.

```javascript
// ❌ BUG: "To be" no necesita auxiliar
console.log("Did you were happy?");         // INCORRECTO
console.log("He didn't was there");         // INCORRECTO

// ✅ FIX: Usa "was/were" directamente
console.log("Were you happy?");             // CORRECTO
console.log("He wasn't there");             // CORRECTO
```
</Aside>

<Aside type="danger" title="Bug #3: Olvidar cambiar la forma del verbo en afirmaciones">
En oraciones afirmativas, NO uses el auxiliar `did`. El verbo va directamente en pasado.

```javascript
// ❌ BUG: No uses "did" en afirmaciones
console.log("I did went to school");        // INCORRECTO
console.log("She did ate breakfast");       // INCORRECTO

// ✅ FIX: Solo usa el verbo en pasado
console.log("I went to school");            // CORRECTO
console.log("She ate breakfast");           // CORRECTO
```
</Aside>
