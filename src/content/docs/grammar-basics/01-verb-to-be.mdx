El verbo **to be** (ser/estar) es el cimiento del inglés. Es como la clase base en programación: casi todo lo demás se construye sobre ella. A diferencia del español, en inglés **un solo verbo** cubre tanto "ser" como "estar", y el contexto determina el significado.

Dominar el verbo to be es esencial porque lo usarás en prácticamente cada conversación, desde presentaciones hasta descripciones de estado y ubicación.

---

## Formas del Verbo To Be (Presente)

El verbo to be tiene tres formas en presente según el sujeto:

```javascript
const toBe = {
  I: "am",           // yo soy/estoy
  you: "are",        // tú eres/estás
  he: "is",          // él es/está
  she: "is",         // ella es/está
  it: "is",          // eso/ello es/está
  we: "are",         // nosotros somos/estamos
  they: "are"        // ellos son/están
};
```

**Ejemplos:**
- I am a student (Soy estudiante / Estoy estudiante)
- You are happy (Eres feliz / Estás feliz)
- She is a doctor (Ella es doctora)
- We are at home (Estamos en casa)
- They are friends (Ellos son amigos)

<Aside type="tip">
**Regla simple:** Usa **am** con "I", **is** con "he/she/it", y **are** con "you/we/they".
</Aside>

## Contracciones

Las contracciones hacen el inglés más natural y fluido. Son casi obligatorias en conversación informal:

<Tabs>
  <TabItem label="Afirmativas">
    ```javascript
    const contractions = {
      "I am": "I'm",           // Yo soy/estoy
      "you are": "you're",     // Tú eres/estás
      "he is": "he's",         // Él es/está
      "she is": "she's",       // Ella es/está
      "it is": "it's",         // Eso es/está
      "we are": "we're",       // Nosotros somos/estamos
      "they are": "they're"    // Ellos son/están
    };
    ```
    
    **Ejemplos:**
    - I'm a developer (Soy desarrollador)
    - You're amazing (Eres increíble)
    - She's from Spain (Ella es de España)
    - We're together (Estamos juntos)
  </TabItem>
  
  <TabItem label="Negativas">
    ```javascript
    const negativeContractions = {
      "am not": "I'm not",        // No soy/estoy (sin otra forma)
      "is not": "isn't",          // No es/está
      "are not": "aren't"         // No son/están
    };
    
    // También puedes contraer el sujeto:
    const alternatives = {
      "he is not": "he's not / he isn't",
      "we are not": "we're not / we aren't"
    };
    ```
    
    **Ejemplos:**
    - I'm not ready (No estoy listo)
    - He isn't here (Él no está aquí)
    - They aren't students (Ellos no son estudiantes)
    - She's not/She isn't happy (Ella no está feliz)
  </TabItem>
</Tabs>

<Aside type="caution">
**Nota sobre "Am not":** No existe la contracción "amn't". Solo puedes decir **"I'm not"**.
</Aside>

## Preguntas con To Be

Las preguntas con to be son muy comunes. Recuerda: **verbo + sujeto**:

```javascript
const questions = {
  identity: "Who are you?",              // ¿Quién eres?
  location: "Where is he?",              // ¿Dónde está él?
  state: "How are you?",                 // ¿Cómo estás?
  reason: "Why are they late?",          // ¿Por qué están tarde?
  time: "What time is it?",              // ¿Qué hora es?
  yesNo: "Is it expensive?"              // ¿Es caro?
};
```

**Respuestas cortas:**

- Are you ready? → Yes, I am / No, I'm not (¿Estás listo? → Sí, lo estoy / No, no lo estoy)
- Is she a doctor? → Yes, she is / No, she isn't (¿Es ella doctora? → Sí, lo es / No, no lo es)
- Are they from Canada? → Yes, they are / No, they aren't (¿Son de Canadá? → Sí, lo son / No, no lo son)

<Aside type="tip">
En respuestas cortas, **no uses contracciones en afirmativas**: "Yes, I **am**" (no "Yes, I'm"). Pero sí puedes usarlas en negativas: "No, I'm not" o "No, I aren't".
</Aside>

## Verbo To Be en Pasado

El verbo to be también tiene formas en pasado: **was** y **were**

```javascript
const toBePast = {
  I: "was",          // yo era/estaba
  you: "were",       // tú eras/estabas
  he: "was",         // él era/estaba
  she: "was",        // ella era/estaba
  it: "was",         // eso era/estaba
  we: "were",        // nosotros éramos/estábamos
  they: "were"       // ellos eran/estaban
};
```

**Ejemplos:**

- I was tired yesterday (Estaba cansado ayer)
- You were right (Tenías razón / Estabas en lo correcto)
- She was a student (Ella era estudiante)
- We were at the park (Estábamos en el parque)
- They were happy (Estaban felices)

**Negativa:** Add "not" después de was/were:
- I wasn't home (No estaba en casa)
- They weren't ready (No estaban listos)

**Pregunta:** Invierte el orden:
- Was he there? (¿Estaba él ahí?)
- Were you busy? (¿Estabas ocupado?)

## Expresiones Comunes con To Be

```javascript
const commonExpressions = {
  age: "I'm 25 years old",           // Tengo 25 años
  feelings: "I'm hungry/thirsty",    // Tengo hambre/sed
  weather: "It's hot/cold/sunny",    // Hace calor/frío/sol
  time: "It's 3 o'clock",            // Son las 3 en punto
  prices: "It's expensive/cheap",    // Es caro/barato
  status: "I'm single/married"       // Soy soltero/casado
};
```

<Aside type="note">
**Importante:** En inglés dices "I'm 25 years old" (literalmente "Soy 25 años viejo"), no "I have 25 years" como en español.
</Aside>

**Más expresiones útiles:**

- I'm afraid (Tengo miedo / Me temo que)
- I'm sure (Estoy seguro)
- It's okay (Está bien)
- It's important (Es importante)
- You're welcome (De nada)
- I'm sorry (Lo siento)

## Errores Comunes

<Aside type="danger">
**Olvidar el verbo to be**

En español puedes decir "Yo estudiante", pero en inglés **siempre** necesitas el verbo:

~~I student~~ / ~~She doctor~~ / ~~We happy~~

→ **I am a student** (Soy estudiante)

→ **She is a doctor** (Ella es doctora)

→ **We are happy** (Estamos felices)
</Aside>

<Aside type="danger">
**Usar "am/is/are" con otros verbos incorrectamente**

El verbo to be NO se combina con otros verbos en presente simple:

~~I am work~~ / ~~She is study~~

→ **I work** (Yo trabajo)

→ **She studies** (Ella estudia)

Excepción: En presente continuo sí se usa: "I **am working**" (Estoy trabajando).
</Aside>

<Aside type="danger">
**Confundir "it's" con "its"**

**It's** = It is (contracción del verbo)

**Its** = posesivo (su, de ello)

Ejemplos:
- **It's** cold today (Está frío hoy) ✓
- The dog lost **its** toy (El perro perdió su juguete) ✓
- ~~The dog lost it's toy~~ ✗
</Aside>

<Aside type="danger">
**Decir "I have X years" para la edad**

En español dices "Tengo 25 años", pero en inglés usas **to be**:

~~I have 25 years~~

→ **I am 25 years old** (Tengo 25 años)

→ **She is 30** (Ella tiene 30)
</Aside>
