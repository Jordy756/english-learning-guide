El presente continuo (present continuous) es como una función que se ejecuta en tiempo real: describe acciones que están ocurriendo en este momento o en un período temporal actual. Se forma con el verbo **to be + verbo-ing**, y es esencial para hablar de lo que está pasando ahora, planes futuros cercanos, o situaciones temporales, siendo uno de los tiempos verbales más usados en conversaciones cotidianas.

---

## Formación del -ing

Reglas para añadir **-ing** al verbo:

<Tabs>
  <TabItem label="Regla normal">
    Añade **-ing** directamente al verbo:
    
    ```javascript
    const regularVerbs = {
      "work": "working",     // trabajar → trabajando
      "play": "playing",     // jugar → jugando
      "read": "reading",     // leer → leyendo
      "eat": "eating",       // comer → comiendo
      "study": "studying"    // estudiar → estudiando
    };
    ```

    **Ejemplos**
    
    - work → **working** (trabajar → trabajando)
    - play → **playing** (jugar → jugando)
    - read → **reading** (leer → leyendo)
    - eat → **eating** (comer → comiendo)
  </TabItem>
  
  <TabItem label="Verbos en -e">
    Quita la **e** final y añade **-ing**:
    
    ```javascript
    const eVerbs = {
      "make": "making",      // hacer → haciendo
      "write": "writing",    // escribir → escribiendo
      "come": "coming",      // venir → viniendo
      "live": "living",      // vivir → viviendo
      "take": "taking"       // tomar → tomando
    };
    ```

    **Ejemplos**
    
    - make → mak~~e~~ → **making** (hacer → haciendo)
    - write → writ~~e~~ → **writing** (escribir → escribiendo)
    - come → com~~e~~ → **coming** (venir → viniendo)
    - live → liv~~e~~ → **living** (vivir → viviendo)
  </TabItem>
  
  <TabItem label="Doblar consonante">
    Si el verbo termina en **consonante + vocal + consonante** y tiene acento en la última sílaba, dobla la última consonante:
    
    ```javascript
    const doubleConsonant = {
      "run": "running",      // correr → corriendo (n→nn)
      "sit": "sitting",      // sentar → sentando (t→tt)
      "swim": "swimming",    // nadar → nadando (m→mm)
      "stop": "stopping",    // parar → parando (p→pp)
      "plan": "planning"     // planear → planeando (n→nn)
    };
    ```

    **Ejemplos**
    
    - run → ru**n** → ru**nn**ing (correr → corriendo)
    - sit → si**t** → si**tt**ing (sentar → sentando)
    - swim → swi**m** → swi**mm**ing (nadar → nadando)
    - stop → sto**p** → sto**pp**ing (parar → parando)
    
    <Aside type="caution" title="Excepciones con w, x, y">
    No se dobla la consonante si el verbo termina en **w**, **x** o **y**:
    - snow → **snowing** ✅ (nevar → nevando)
    - fix → **fixing** ✅ (arreglar → arreglando)
    </Aside>
  </TabItem>
  
  <TabItem label="Verbos en -ie">
    Cambia **-ie** por **-ying**:
    
    ```javascript
    const ieVerbs = {
      "lie": "lying",        // mentir/yacer → mintiendo
      "die": "dying",        // morir → muriendo
      "tie": "tying"         // atar → atando
    };
    ```

    **Ejemplos**
    
    - lie → l~~ie~~ → l**y**ing (mentir → mintiendo)
    - die → d~~ie~~ → d**y**ing (morir → muriendo)
    - tie → t~~ie~~ → t**y**ing (atar → atando)
  </TabItem>
</Tabs>

## Expresiones de tiempo

<Tabs>
  <TabItem label="Ahora mismo">
    ```javascript
    const nowExpressions = [
      "now",               // ahora
      "right now",         // ahora mismo
      "at the moment",     // en este momento
      "at present",        // en el presente
      "currently"          // actualmente
    ];
    ```

    **Ejemplos**

    - I **am working** **now** (Estoy trabajando ahora)
    - She **is cooking** **right now** (Ella está cocinando ahora mismo)
    - They **are studying** **at the moment** (Están estudiando en este momento)
    - He **is sleeping** **at present** (Él está durmiendo actualmente)
  </TabItem>
  
  <TabItem label="Situación temporal">
    ```javascript
    const temporaryExpressions = [
      "these days",        // en estos días
      "this week",         // esta semana
      "this month",        // este mes
      "this year",         // este año
      "currently",         // actualmente
      "temporarily",       // temporalmente
      "for the moment",    // por el momento
      "for now"            // por ahora
    ];
    ```

    **Ejemplos**

    - I **am learning** Spanish **these days** (Estoy aprendiendo español en estos días)
    - She **is working** from home **this week** (Está trabajando desde casa esta semana)
    - We **are staying** at a hotel **this month** (Nos quedamos en un hotel este mes)
    - They **are living** in Tokyo **temporarily** (Están viviendo en Tokio temporalmente)
    - He **is using** my car **for now** (Está usando mi carro por ahora)
  </TabItem>
  
  <TabItem label="Planes futuros">

    ```javascript
    const futureExpressions = [
      "tomorrow",          // mañana
      "tonight",           // esta noche
      "this evening",      // esta tarde/noche

      "next week",         // la próxima semana
      "next month",        // el próximo mes
      "next year",         // el próximo año
      "on Monday",         // el lunes
      "in the morning",    // en la mañana

    ];
    ```

    **Ejemplos**
        
    - I **am meeting** Sarah **tomorrow** (Me encuentro con Sarah mañana)
    - We **are going** to the cinema **tonight** (Vamos al cine esta noche)
    - She **is coming** over **this evening** (Ella viene esta tarde)
    - They **are traveling** to Spain **next week** (Viajan a España la próxima semana)
    - He **is starting** his new job **on Monday** (Empieza su nuevo trabajo el lunes)
  </TabItem>
  
  <TabItem label="Cambio progresivo">

    ```javascript
      "more and more",     // cada vez más
      "increasingly",      // cada vez más/crecientemente

      "slowly",            // lentamente

      "constantly",        // constantemente
      "steadily",          // constantemente/establemente
    ```

    **Ejemplos**

    - The weather **is getting** colder **gradually** (El clima se está poniendo más frío gradualmente)

## Estructura del Presente Continuo

<Tabs>
  <TabItem label="Afirmativa">
    ```javascript
    const presentContinuous = {
      structure: "Subject + to be + verb-ing",
      examples: [
        "I am working",           // Estoy trabajando
        "You are reading",        // Estás leyendo
        "He is sleeping",         // Él está durmiendo
        "She is cooking",         // Ella está cocinando
        "It is raining",          // Está lloviendo
        "We are studying",        // Estamos estudiando
        "They are playing"        // Están jugando
      ]
    };
    ```
    
    **Patrón:** Usa la forma correcta de **to be** (am/is/are) + verbo con **-ing**.
    
    - I **am eating** lunch (Estoy almorzando)
    - She **is watching** TV (Ella está viendo televisión)
    - We **are learning** English (Estamos aprendiendo inglés)
  </TabItem>
  
  <TabItem label="Negativa">
    ```javascript
    const negative = {
      structure: "Subject + to be + not + verb-ing",
      examples: [
        "I am not working",         // No estoy trabajando
        "You are not reading",      // No estás leyendo
        "He is not sleeping",       // Él no está durmiendo
        "She is not cooking",       // Ella no está cocinando
        "It is not raining",        // No está lloviendo
        "We are not studying",      // No estamos estudiando
        "They are not playing"      // No están jugando
      ]
    };
    ```
    
    Para negar, añade **not** después de **to be**.
    
    - I **am not listening** (No estoy escuchando)
    - He **is not coming** today (Él no viene hoy)
    - They **are not working** now (Ellos no están trabajando ahora)
  </TabItem>
  
  <TabItem label="Pregunta">
    ```javascript
    const question = {
      structure: "To be + subject + verb-ing + ?",
      examples: [
        "Am I working?",              // ¿Estoy trabajando?
        "Are you reading?",           // ¿Estás leyendo?
        "Is he sleeping?",            // ¿Está él durmiendo?
        "Is she cooking?",            // ¿Está ella cocinando?
        "Is it raining?",             // ¿Está lloviendo?
        "Are we studying?",           // ¿Estamos estudiando?
        "Are they playing?"           // ¿Están jugando?
      ]
    };
    ```
    
    Invierte el orden: **to be** antes del sujeto.
    
    **Respuestas cortas:**
    - **Are** you working? → Yes, I **am** / No, I **am** not
    - **Is** she coming? → Yes, she **is** / No, she **is** not
  </TabItem>
</Tabs>

## Verbos de Estado (Stative Verbs)

Algunos verbos describen **estados o condiciones** en lugar de acciones, y normalmente NO se usan en presente continuo porque expresan situaciones permanentes sin progreso visible.

<Tabs>
  <TabItem label="Mentales">
    ```javascript
    const mentalStateVerbs = {
      knowledge: ["know", "understand", "realize"],
      // ["saber", "entender", "darse cuenta"]

      belief: ["believe", "think", "doubt", "suppose"],
      // ["creer", "pensar", "dudar", "suponer"]

      memory: ["remember", "forget", "recall"],
      // ["recordar", "olvidar", "rememorar"]
      
      mental: ["recognize", "mean", "imagine"]
      // ["reconocer", "significar/tener intención", "imaginar"]
    };
    ```

    **Ejemplos**
    
    - I **know** the answer (Sé la respuesta)
    - She **understands** English (Ella entiende inglés)
    - We **believe** you (Te creemos)
    - He **remembers** your name (Él recuerda tu nombre)
    - They **recognize** this place (Reconocen este lugar)
    - I **doubt** it (Lo dudo)
    - She **means** well (Ella tiene buenas intenciones)
  </TabItem>
  
  <TabItem label="Emociones">
    ```javascript
    const emotionalStateVerbs = {
      positive: ["love", "like", "adore", "enjoy", "prefer"],
      // ["amar", "gustar", "adorar", "disfrutar", "preferir"]
      
      negative: ["hate", "dislike", "detest"],
      // ["odiar", "disgustar", "detestar"]
      
      desire: ["want", "need", "wish", "desire"],
      // ["querer", "necesitar", "desear", "desear"]
      
      attitude: ["care", "mind", "appreciate"]
      // ["importar", "importar/molestar", "apreciar"]
    };
    ```

    **Ejemplos**
    
    - I **love** chocolate (Amo el chocolate)
    - She **likes** coffee (A ella le gusta el café)
    - We **hate** waiting (Odiamos esperar)
    - He **wants** a new phone (Él quiere un teléfono nuevo)
    - They **need** help (Necesitan ayuda)
    - I **prefer** tea (Prefiero té)
    - She **appreciates** your effort (Ella aprecia tu esfuerzo)
  </TabItem>
  
  <TabItem label="Sentidos">
    ```javascript
    const sensoryStateVerbs = {
      perception: ["see", "hear", "smell", "taste", "feel"],
      // ["ver", "oír", "oler", "saber (gusto)", "sentir (tacto)"]
      
      appearance: ["look", "sound", "seem", "appear"]
      // ["parecer/verse", "sonar", "parecer", "aparecer"]
    };
    ```

    **Ejemplos**
    
    - I **see** the problem (Veo el problema)
    - She **hears** music (Ella escucha música)
    - It **smells** good (Huele bien)
    - This **tastes** delicious (Esto sabe delicioso)
    - He **looks** tired (Él se ve cansado)
    - That **sounds** interesting (Eso suena interesante)
    - They **seem** happy (Parecen felices)
  </TabItem>
  
  <TabItem label="Posesión">
    ```javascript
    const possessionStateVerbs = {
      ownership: ["have", "own", "possess", "belong"],
      // ["tener", "poseer", "poseer", "pertenecer"]
      
      relationship: ["contain", "include", "consist", "depend"]
      // ["contener", "incluir", "consistir", "depender"]
    };
    ```

    **Ejemplos**
    
    - I **have** a car (Tengo un carro)
    - She **owns** a house (Ella posee una casa)
    - This **belongs** to me (Esto me pertenece)
    - The box **contains** books (La caja contiene libros)
    - It **includes** breakfast (Incluye desayuno)
    - Success **depends** on effort (El éxito depende del esfuerzo)
  </TabItem>
  
  <TabItem label="Otros">
    ```javascript
    const otherStateVerbs = {
      existence: ["be", "exist", "matter"],
      // ["ser/estar", "existir", "importar"]
      
      measurement: ["weigh", "measure", "cost"],
      // ["pesar", "medir", "costar"]
      
      relation: ["equal", "resemble", "differ", "fit"]
      // ["ser igual a", "parecerse a", "diferir", "quedar bien"]
    };
    ```

    **Ejemplos**
    
    - This **is** important (Esto es importante)
    - Ghosts **exist** (Los fantasmas existen)
    - It **matters** to me (Me importa)
    - I **weigh** 70 kilos (Peso 70 kilos)
    - This **costs** too much (Esto cuesta demasiado)
    - She **resembles** her mother (Ella se parece a su madre)
    - These shoes **fit** perfectly (Estos zapatos quedan perfectos)
  </TabItem>
</Tabs>

<Aside type="note" title="Por qué no usan -ing estos verbos">
Estos verbos describen **estados permanentes o de larga duración**, no acciones momentáneas. Por ejemplo, "love" es un sentimiento constante, no algo que "estás haciendo" temporalmente. En inglés, se usan en presente simple porque describen **cómo son las cosas**, no **qué está pasando**.
</Aside>

<Aside type="tip" title="Excepciones importantes">
Algunos verbos de estado pueden usarse en continuo cuando cambian de significado:

**Have:**
- I **have** a car (Tengo un carro - posesión) → simple ✅
- I **am having** lunch (Estoy almorzando - acción) → continuo ✅
- We **are having** a party (Estamos haciendo una fiesta - evento) → continuo ✅

**Think:**
- I **think** it's good (Creo que es bueno - opinión) → simple ✅
- I **am thinking** about it (Estoy pensándolo - proceso activo) → continuo ✅

**See:**
- I **see** the problem (Veo/entiendo el problema) → simple ✅
- I **am seeing** a doctor tomorrow (Voy a ver a un doctor - cita) → continuo ✅
</Aside>

## Errores Comunes

<Aside type="danger" title="Olvidar el verbo to be">
El presente continuo SIEMPRE necesita **to be**:

- I working now / She studying English ⛔
- I **am working** now (Estoy trabajando ahora) ✅
- She **is studying** English (Ella está estudiando inglés) ✅
</Aside>

<Aside type="danger" title="Error en la formación del -ing">
Sigue las reglas de ortografía correctamente:

- runing / writting / stoping / swiming ⛔
- **running** (dobla n) ✅
- **writing** (quita e) ✅
- **stopping** (dobla p) ✅
- **swimming** (dobla m) ✅
</Aside>

<Aside type="danger" title="Usar verbos de estado con -ing">
Los verbos de estado normalmente NO van en continuo:

- I am knowing / She is loving / They are having a car ⛔
- I **know** (Sé) ✅
- She **loves** (Ella ama) ✅
- They **have** a car (Tienen un carro) ✅

Excepción: "I am having lunch" (acción, no posesión) ✅
</Aside>

<Aside type="danger" title="Confundir usos con presente simple">
Usa presente simple para rutinas, continuo para acciones temporales:

- I am working every day (rutina con continuo) ⛔
- I **work** every day (Trabajo todos los días - rutina) ✅
- I **am working** now (Estoy trabajando ahora - temporal) ✅
</Aside>
